34. mohon di jawab dana cara kerjanya giman..??Nilai Q1, Q2, dan Q3 dari sekumpulan data : 5, 4, 2, 6, 7, 9, 8 dan 10, adalah….a. Q1=4,5 Q2=6,0 Q3=8,0b. Q1=4,5 Q2=6,5 Q3=8,5c. Q1=4,5 Q2=7,0 Q3=9,0d. Q1=4,5 Q2=5,0 Q3=9,0


Urutkan data dari terkecil hingga terbesar
2, 4, 5, 6, 7, 8, 9, 10

① Kuartil atas (Q3) → 3/4 (n + 1)
= 3/4 (8 + 1)
= 3/4 (9)
= 6,75

Maka Q3 = Data ke6 + 0,75(Data ke7 - Data ke6)
= 8 + 0,75(9 - 8)
= 8,75

② Kuartil bawah (Q1) → 1/4 (n + 1)
= 1/4 (8+ 1)
= 1/4 (9)
= 2,25

Maka Q1 = Data ke2 + 0,25(Data ke3 - Data ke2)
= 4 + 0,25(5 - 4)
= 4,25

③ Kuartil tengah (Q2) → 2/4 (n + 1)
= 2/4 (8 + 1)
= 2/4 (9)
= 4,5

Maka Q2 = Data ke4 + 0,5(Data ke5 - Data ke4)
= 6 + 0,5(7 - 6)
= 6,5

=> x1, x2, ...., x7

jumlah data n = 7

[tex] Q_1 = x_{\frac{n + 1}{4}} = x_{\frac{7 + 1}{4}} = x_2 \\ \huge \boxed{\boxed{ Q_1 = 7}} [/tex]

[tex] Q_2 = x_{\frac{n + 1}{2}} = x_{\frac{7 + 1}{2}} = x_4 \\ \huge \boxed{\boxed{ Q_2 = 8}} [/tex]

[tex] Q_3 = x_{\frac{3(n + 1)}{4}} = x_{\frac{3(7 + 1)}{4}} = x_6 \\ \huge \boxed{\boxed{ Q_3 = 9}} [/tex]
