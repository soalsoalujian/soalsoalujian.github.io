Diketahui:
Harga Beli = 12.000.000
Harga Jual= 11.000.700

ditanya:
% kerugian=?

Jawab:
Besar kerugian = Harga beli - harga jual
                         = 12.000.000 - 11.000.700
                         = 999.300

% kerugian = (besar kerugian / harga beli) x 100%
                   = (999.300 / 12.000.000) x 100%
                   = 0,083275 x 100%
                   = 8,3275%
