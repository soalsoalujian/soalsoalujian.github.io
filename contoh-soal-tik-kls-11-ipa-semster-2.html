Input: Angka bulat dalam satuan jam

Output:

Tampilan detik dari angka yang diinputTampilan menit dari angka yang diinput

Rumus untuk mengubah jam menjadi menit dan detik:

Menit = Jam * 60

Detik = Menit * 60

Berikut adalah pseudocode untuk mengkonversi jam menjadi detik dan menit

BEGIN

  hour = Get User Input

  minute = hour * 60

  second = minute * 60

  Print hour + ' jam sama dengan ' + minute + ' menit'

  Print hour + ' jam sama dengan ' + second + ' detik'

END
