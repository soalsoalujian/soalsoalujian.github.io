22. (1105+2105+....+10105)÷25 brpakah hasilnya?​


Jawab:

2242

Penjelasan dengan langkah-langkah:

Diketahui

a = 1105

b = 1000

Un = a + (n-1)b

10105 = 1105 + (n-1)1000

10105 = 1105 + 1000n - 1000

10105 = 105 + 100n

10000 = 100n

n = 10

Ditanya

S10 / 25

Jawab

Sn = n/2 (2a + (n-1)b)

S10 = 10/2 (2(1105) + (10-1)(1000))

S10 = 5 (2210 + 9000)

S10 = 5 (11210)

S10 = 56050

S10 / 25

= 56050 / 25

= 2242
