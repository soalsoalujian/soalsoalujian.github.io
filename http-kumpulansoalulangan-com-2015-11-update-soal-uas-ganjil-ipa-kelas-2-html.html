15. Bagaimana proses terjadinya menstruasi? Jelaskan! [Soal Ujian Akhir Semester UAS 2015 Kelas 9]


proses terjadinya menstruasi terjadi dlm 3 fase. pertama fase menstruasi pada fase ini hormon FSH (Follicle Stimulating Hormone) memicu berkembangnya folikel dlm ovarium. pda fase ini dinding rahim meluruh sehingga wanita mengalami menstruasi, fase kedua yaitu fase proliferasi terjadi penebalan dinding rahim yg di sebabkan oleh hormon estrogen dan progesteron, fase ketiga yaitu fase sekretori terjadi folikel akan melepaskan korpus luteum . jika pda fase ini sel telur tdkdi buahi sel sperma maka akan dikirimkan sinyal kpd korpus luteum sehingga tdk memproduksi hormon estrogen dan progestero lagi dan wanita tsb pun akan mengalami menstruasi lagi

18. Jelaskan proses terjadinya menstruasi! [Soal Ujian Akhir Semester UAS 2015 Kelas 9]


menstruasi terjadi apabila pendarahan yg terjadi akibat luruhnya dinding sebelah dalam rahim (endometrium) yg banyak. mengandung pembuluh darahkelenjar Hipofisis di dalam otak melepaskan hormon FSH (follicle stimulating hormone) membuat sel telur tumbuh di dalam ovarium. hormon estrogen bekerja sama dengan FSH membantu sel telur tumbuh kemudian memberi signal kepada rahim agar mempersiapkan diri untuk menerima sel telur tersebut.
ketika sel telur matang otak melepaskan hormon LH (luteinizing hormone), jika sel telur tidak dibuahi maka endometrium akan meluruh dan terjadi proses menstruasi

http://www.sekolahku.com/informasi/pendaftaran.html

1. Protokol yang digunakan adalah HTTP (Hypertext Transfer Protocol)
2. Top Level Domain = .com
3. Second Level Domain = sekolahku
3. Path domain = informasi
4. Nama file = pendaftaran.html

• Jawaban panjang: 

Domain tersebut menggunakan protokol HTTP (Hypertext Transfer Protocol). Mengunakan Top Level Domain (TLD) .com yang biasa digunakan untuk komersial / perusahaan dan Second Level Domain (SLD) sekolahku. Path pada domain tersebut adalah "informasi", dan Nama filenya adalah "pendaftaran.html"

Ovulasi adalah proses terlepasnya sel ovum dari ovarium sebagai akibat pecahnya folikel yang telah masak.
