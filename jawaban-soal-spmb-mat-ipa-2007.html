8.) dari gambar didapatkan sin2x = 4/5
  
     sin 2x ⇒ 2 sin x. cos x = 4/5
 
                      sin x. cos x = 2/5
 
     Misalkan pada segmen BC ada titik D, misalkan BD = p dan DC = 4-p. 

     didapatkan, sin x . cos x = 2/5   ⇒   3p / p² + 9 = 2/5
  
                                                       ⇔  2p² - 15p + 18 = 0
  
                                                       ⇔ (2p - 3)(p - 6) = 0

                                                         p = 3/2 atau p = 6
       Sehingga tan x = p/3 
  
       Maka tan x₁ = 1/2 dan tan x₂ = 2
  
         tan x₁ - 1/2 = 0 dan tan x₂ - 2 = 0
            
               
                 (tan x₁ - 1/2)(tan x₂ - 2) = 0

                 (2tan x₁ - 1)(tan x₂ - 2) = 0

                 (2tan x - 1)(tan x - 2) = 0
 
                    2tan²x - 5tan x + 2 = 0
 
 
 ∴ Persamaan trigonometri yang berkaitan adalah  2tan²x - 5tan x + 2 = 0 (A)

[tex]\pi {r}^{2} t \: = \frac{22}{7} \: \times 140 cm \: \times 140 \: cm \times 200 \: cm = 12.320.000 \: cm \\ = 12.320 \: liter[/tex]

Penjelasan dengan langkah-langkah:

jadi jawabannya: 12.320 L

Diketahui sebuah bak mandi berbentuk tabung dengan jari jari 140 cm dan tinggi 200 cm. Daya tampung air pada bak tersebut adalah 12.320.000 cm³/ 12.320 liter.

PENYELESAIAN SOAL.Diketahui :Bangun = TabungJari jari (r) = 140 cmTinggi (t) = 200 cm

Ditanyakan :Daya tampung/Volume

Jawaban :

Rumus Volume Tabung :

[tex]\boxed{\sf{V = \pi \times r^2 \times t}}[/tex]

.

[tex]\sf{V = \pi \times r^2 \times t}[/tex]

[tex]\sf{V = \frac{22}{7} \times 140^2 \times 200 }[/tex]

[tex]\sf{V = \frac{22}{7}\times \:19600\times \:200}[/tex]

[tex]\bold{V = 12.320.000 \:cm^3}[/tex]

KESIMPULAN.

Jadi daya tampung tabung tersebut adalah 12.320.000 cm³/ 12.320 liter.

38. bagaimana cara menjawab soal mat detik2 ini ? tolong bantuan nya ! [mat]​


Dita : Nisa

5 : 9

45 cm : ?

Nisa = [Pita Dita : Perbandingan Dita] x Perbandingan Nisa

Nisa = [45 cm : 5] x 9

Nisa = 9 x 9

Nisa = 81 cm

Selisih = Nisa - Dita

= 81 - 45

= 36 cm [A]

Jawab:

Penjelasan dengan langkah-langkah:

Dita : 45 cm ( 5 )  

Nisa : 45 ÷ 5 = 9

         9 × 9 (perbandingan) = 81 cm

Selisih pita =  81 - 45

                 = 36 Cm ( A )
