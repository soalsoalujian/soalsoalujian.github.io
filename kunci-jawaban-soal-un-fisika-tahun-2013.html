clik gambar biar liat penuh :)Diket dan dit:
*n NaOH = 100 mL x 0,008 M = 0.8 mmol
*n CH3COOH = 100 mL x 0,008 M = 0,8 mmol
*Ka = 10^-5
*Ksp Fe(OH)2 = 1,6 x 10^-15
*[Fe2+] = ? M

A. Mencari nilai [OH-]
--> Soal UN ini mengaitkan 2 bab (kyk SBMPTN, ya), yaitu hidrolisis garam dan hasil kali kelarutan (Ksp), sehingga:
     CH3COOH + NaOH --> CH3COONa + H2O
m        0.8           0,8
b        -0.8          -0.8              +0.8          +0.8
s           -              -                 0.8

[G] = 0.8 mmol / 200 mL = 4 x 10^-3

Bersisa garamnya sebesar 0,8 mmol, masukkan ke rumus hidrolisis:
[OH-] = √Kw x [G]/Ka
         = √10^-14 x 4 x 10^-3 / 10^-5
         = 2 x 10^-6

B. Mencari [Fe2+] dari Fe(OH)2
--> Untuk mencarinya, buat reaksi ionisasi Besi (II) hidroksida tsb:
Fe(OH)2 ⇄ Fe2+ + 2OH-

Ksp = [Fe2+] [OH-]^2
1,6 x 10^-15 = [Fe2+] [2 x 10^-6]^2
[Fe2+] = 1,6 x 10^-15 / 4 x 10^-12
           = 0.4 x 10^-3
           = 4 x 10^-4 Molar (D)

27. kunci jawaban ipa fisika kelas 7 hal 152 kurikulum 2013​


Kabel Listrik dipasang agak kendur agar kabel tidak putus saat suhu turun dan terjadi penyusutan.Koefisien muai panjang besi sebesar 12 x 10⁻⁵/°C atau  1,2 x 10⁻⁶/°CPanjang Baja pada suhu 70°C adalah 40,04 cm

Pembahasan

Pemuaian

Saat benda dikenai kalor, maka akan terjadi pemuaian pada benda tersebut yang akan mengakibatkan pertambahan ukuran pada benda. dan sebaliknya saat kalor dilepaskan dari benda, akan terjadi penyusutan, besarnya pemuaian atau penyusutan tergantung pada koefisien muai benda dan perubahan suhu.

Pada Pemuaian Panjang berlaku persamaan :ΔL = Lo x α x Δt  Lt = Lo + ΔL  

atau  

Lt = Lo + (Lo x α x Δt)  

Keterangan :  

Lo = Panjang awal = m  

Lt = Panjang akhir = m  

ΔL = Pertambahan Panjang = m  

α = Koefisien muai panjang

Pada Pemuaian Luas berlaku persamaan :ΔA = Ao x β x Δt  

dan  

At = Ao + ΔA  

Atau  

At = Ao + (Ao x β x Δt)  

Keterangan :  

ΔA = Pertambahan Luas  

Ao = Luas awal  

At = Luas Akhir  

β = Koefisien Muai Luas  

β = 2 α  

Pada Pemuaian Volume berlaku persamaan :ΔV = Vo x γ x Δt  

dan untuk Volume akhir adalah :  

Vt = Vo + ΔV  

atau  

Vt = V0 + (V0 x γ x Δt)  

Keterangan  

ΔV = Pertambahan Volume  

γ = Koefisien Muai Ruang atau Volume  

besarnya γ = 3α  

Δt= Perubahan Suhu  

Vo= Volume awal  

Vt = Volume akhir  

Pengerjaan soal :

Soal Np 1 :

Pemasangan Kabel Listrik dibuat kendur agar saatterjadi penyusutan dimalam hari kabel tidak putus, seperti sudah dijelaskan diatas jika benda mendapatkan kalor akan mengalami pemuaian, dan jika benda melepaskan kalor (mengalami penurunan suhu) akan terjadi penyuutan. Perubahan panjang tersebut dirumuskan dalam persamaan :

ΔL = Lo x α x Δt

Keterangan :  

Lo = Panjang awal = m  

ΔL = Pertambahan Panjang = m  

α = Koefisien muai panjang =  

Soal No 2 :Diketahui :

t1 = Suhu Awal = 0°C

t2 = Suhu akhir = 100°C

Lo = Panjang awal besi = 200 cm

Lt = Panjang akhir besi = 200,24 cm

ΔL = Perubahan panjang besi = 200,24 cm - 200 cm = 0,24 cm

Ditanyakan :

α = Kofisien muai panjang besi = ...

Penyelesaian :

ΔL = Lo x α x Δt

0,24 = 200 x α x 100

0,24 = 2.000α

α = 0,24 : 2.000

α = (24 x 10⁻²) : (2 x 10³)

α = 12 x 10⁻⁵   = 1,2 x 10⁻⁶

Soal No 3 :Diketahui :

t1 = Suhu awal = 20°C

t2 = Suhu akhir = 70°C

Δt = Perubahan Suhu = 70 - 20 = 50°C

Lo = Panjang Baja mula - mula = 40 cm

α   = Koefisien muai baja = 0,00002/°C = 2 x 10⁻⁵/°C

Jawab:

Barisan bilangan 5, 11, 17, 23, 29, ... merupakan barisan aritmetika

U₁ = a = 5

U₂ = a + b = 11

b = 11 - 5 = 6


[tex]U_{n} = a + (n-1)b\\U_{50} = 5 + (50-1)6\\U_{50} = 5 + 49.6\\U_{50} = 299[/tex]
