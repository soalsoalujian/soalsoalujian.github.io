16. Soal dibawah ini carilah koefiaisennya a. P1 = 1.000 Q1 = 4.000 P2 = 1.200 Q2 = 2000 b. P1 = 800 Q1 = 6.000 P2 = 1.000 Q2 = 4.000 c. P1 = 1.000 Q1 = 5.000 P2 = 800 Q2 = 6.000 d. P1 = 15.000 Q1 = 20.000 P2 = 10.000 Q2 = 18.000


Berdasarkan soal tersebut maka hasil perhitungan di dapat koefisien fungsi penawaran dari pertanyaan pertanyaan tersebut ialah :

a. Ps = 1.400 - 0,1Q

b. Ps = 1.400 - 0,1Q

c. Ps = 2.000 - 0,5Q

d. Ps = 2,5Q - 35.000

Penjelasan dengan langkah-langkah:

Untuk mencari fungsi permintaan atau pun fungsi penawaran adalah dengan menggunakan rumus sebagai berikut:

(P-P1)/(P2-P1)=(Q-Q1)/(Q2-Q1)  atau  P-P1 = m(Q-Q1)

Ket :

m = gradien

P1 = harga awal

P2 = harga setelah mengalami perubahan

Q1 = kuantias awal

Q2 = kuantitas setelah mengalami perubahan

Diketahui:

a. P1 = 1.000, Q1 = 4.000, P2 = 1.200, Q2 = 2000

b. P1 = 800, Q1 = 6.000, P2 = 1.000, Q2 = 4.000

c. P1 = 1.000, Q1 = 5.000, P2 = 800, Q2 = 6.000

d. P1 = 15.000, Q1 = 20.000, P2 = 10.000, Q2 = 18.000

Ditanya:

Carilah nilai koefisien Fungsi Penawaran?

Penyelesaian:

a. P1 = 1.000, Q1 = 4.000, P2 = 1.200, Q2 = 2000

(P-P1)/(P2-P1)=(Q-Q1)/(Q2-Q1)

(P- 1.000)/( 1.200-1.000)=(Q-4.000)/(2000-4.000)

(P- 1.000)/200 = (Q-4.000)/-2.000

-2.000P + 2.000.000 = 200Q - 800.000

-2.000P = 200Q - 2.800.000

P = 1.400 - 0,1Q

b. P1 = 800, Q1 = 6.000, P2 = 1.000, Q2 = 4.000

(P-P1)/(P2-P1)=(Q-Q1)/(Q2-Q1)

(P- 800)/( 1.000-800)=(Q-6.000)/(4.000-6.000)

(P- 800)/200 = (Q-6.000)/-2.000

-2.000P + 1.600.000 = 200Q - 1.200.000

-2.000P = 200Q - 2.800.000

P = 1.400 - 0,1Q

c. P1 = 1.000, Q1 = 5.000, P2 = 800, Q2 = 6.000

(P-P1)/(P2-P1)=(Q-Q1)/(Q2-Q1)

(P- 1.000)/(800-1.000)=(Q-5.000)/(6.000-5.000)

(P- 1.000)/-200 = (Q-5.000)/1.000

1.000P - 1.000.000 = - 200Q + 1.000.000

1.000P = -200Q + 2.000.000

P = 2.000 - 0,5Q

d. P1 = 15.000, Q1 = 20.000, P2 = 10.000, Q2 = 18.000

(P-P1)/(P2-P1)=(Q-Q1)/(Q2-Q1)

(P- 15.000)/(10.000-15.000)=(Q-20.000)/(18.000-20.000)

(P- 15.000)/-5.000 = (Q-20.000)/-2.000

-2.000P + 30.000.000 = - 5.000Q + 100.000.000

-2.000P = - 5.000Q + 70.000.000

P = 2,5Q - 35.000

20. Tugas ipa pelajari dari buku pkt hlmn 138 persilangan dihibrida (dua sifat beda). Buatlah persilangan di bawah ini! #parental (p1) : Rambut hitam kriting (HHkk) x rambut pirang lurus (hhkk) #Gamet: Hk x hk #Filial: (p1) : Hh Kk ​


Jawaban:

Dalam persilangan dihibrida antara induk dengan fenotipe rambut hitam kriting (HHkk) x rambut pirang lurus (hhkk), kita akan memperhatikan dua sifat yang berbeda, yaitu warna rambut (hitam atau pirang) dan jenis rambut (kriting atau lurus).

Induk p1:

Rambut hitam kriting (HHkk)

Rambut pirang lurus (hhkk)

Gamet p1:

Hk (dari induk dengan genotip HHkk)

hk (dari induk dengan genotip hhkk)

Ketika kedua gamet ini bergabung, kita akan mendapatkan kombinasi genotip pada filial (f1) yaitu Hhkk dan hhkk.

Filial f1:

Hhkk (rambut hitam kriting)

hhkk (rambut pirang lurus)

Dalam persilangan ini, kita melihat segregasi genetik pada dua sifat yang berbeda, yaitu warna rambut dan jenis rambut. Keturunan memperoleh kombinasi genotip yang menggabungkan sifat-sifat dari kedua induk, sehingga muncul variasi fenotipik dalam filial berupa kombinasi genotip Hhkk (rambut hitam kriting) dan hhkk (rambut pirang lurus).
