40. mohon bantuannya di lumpulkan besok​


Penjelasan dengan langkah-langkah:

1.

• Bangun 1 merupakan 4 buah persegi yang terdapat 4 buah ¼ lingkaran didalamnya, maka (Lb = Luas Bangun yang Diarsir)

Lb = 4(Luas Persegi) - 4(Luas ¼ Lingkaran)

Lb = 4s² - 4(¼ πr²)

Lb = 4s² - πr² ... (karena panjang sisi = jari², maka)

Lb = 4s² - πs²

Lb = (4 - π) s²

Lb = (4 - (22/7)) 7²

Lb = 196 - 154 = 42 cm²

• Kb = Keliling bangun = Keliling 4 persegi

Kb = 4s

Kb = 4(14) = 56 cm

2.

• Bangun 2 merupakan 1 buah persegi panjang dan 2 buah½ lingkaran, maka (Lb = Luas Bangun yang Diarsir)

Lb = (Luas Persegi Panjang) + 2(Luas ½ Lingkaran)

Lb = p×l + 2(½ πr²)

Lb = (p×l) + (πr²)

Lb = (28 × 12) + ( (22/7) (14)² )

Lb = 336 + 616

Lb = 952 cm²

• Kb = Keliling bangun = (Keliling persegi panjang) + 2(½ Keliling lingkaran)

Kb = 2(p + l) + 2(½ × 2πr)

Kb = 2(12) + 2πr

Kb = 24 + 2 (22/7) (14)

Kb = 24 + 88 = 112 cm

3.

• Bangun 3 merupakan 1 buah persegi dan 1 buah ¾ lingkaran, maka (Lb = Luas Bangun yang Diarsir)

Lb = (Luas Persegi) + (Luas ¾ Lingkaran)

Lb = s² + (¾ πr²)

Lb = (14²) + ( ¾ (22/7) (14)² )

Lb = 196 + (¾ 616)

Lb = 196 + 462 = 658 cm²

• Kb = Keliling bangun = (Keliling persegi) + (¾ Keliling lingkaran)

Kb = 4s + (¾ × 2πr)

Kb = 4(7) + ¾ (2 × (22/7) × 14)

Kb = 28 + ¾ (88)

Kb = 28 + 66 = 94 cm

4.

• Bangun 4 merupakan 4 buah persegi dan 2 buah ¼ lingkaran, maka (Lb = Luas Bangun yang Diarsir)

Lb = 4(Luas Persegi) - 2(Luas ¼ Lingkaran)

Lb = 4s² - 2(¼ πr²)

Lb = 4s² - (½ πr²)

Lb = 4(10²) - ( ½ (22/7) (10)² )

Lb = 400 - (1100/7)

Lb = 2800/7 - 1100/7 = 1700/7 cm² = 243 cm³

• Kb = Keliling bangun = (Keliling 4 persegi)

Kb = (4s)

Kb = 4(20) = 80 cm

5.

• Bangun 5 merupakan 1 buah persegi dan 4 buah ½ lingkaran, maka (Lb = Luas Bangun yang Diarsir)

Lb = (Luas Persegi) - (Luas ½ Lingkaran)

Lb = s² - (½ πr²)

Lb = (42²) - ( ½ (22/7) (42/3)² )

Lb = 1764 - (½ 616)

Lb = 1764 - 308 = 1456 cm²

• Kb = Keliling bangun = (Keliling persegi)

Kb = 4s

Kb = 4(42) = 168 cm
