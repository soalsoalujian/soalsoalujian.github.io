Keterangan: x = simpangan (m)

                     l = panjang tali

                     m = massa (kg)

                     g = percepatan gravitasi (m/s²)

  = 0,5 · 9,8 N

      = 100 · 0,05 N

   (2) Periode centilever 0,4 s -> SALAH, periode centilever = 0,8 s

   (3) Frekuensi centilever 1,25 Hz -> BENAR, f = 1/T = 1/0,8 = 1,25 Hz

   (4) Saat t = 1,8 s, centilever melakukan 2,25 getaran -> BENAR

        f = n/t

  1,25 = n/1,8

       n = 1,25 · 1,8 = 2,25

Pernyataan yang tepat adalah pernyataan nomor (1), (3), dan (4).

4. F = mg

      = 4 · 10 N

      = 40 N

  EP = 1/2 · k · x²

       = 1/2 · (k · x) · x

       = 1/2 · F · x

       = 1/2 · 40 · 0,2 J

       = 4 J

5. Misalkan percepatan gravitasi di planet tersebut adalah g, dan percepatan gravitasi di Bumi adalah g'.

g = 1/4 · g'

[tex]T=2\pi\sqrt{\frac{l}{g}}\\ 2 = 2\pi\sqrt{\frac{l}{g} } \\1 = \pi\sqrt{\frac{l}{g} } \\1 = \frac{\pi^2l}{g} \\g = \pi^2l[/tex]             [tex]T'=2\pi\sqrt{\frac{l}{g'} } \\T' = 2\pi\sqrt{\frac{l}{4g} } \\T'^2 = 4\pi^2\frac{l}{4g} \\T'^2 = \frac{\pi^2l}{g} \\T'^2 = \frac{g}{g} = 1\\T' = 1 s[/tex]  

Periode jam bandul tersebut di bumi adalah 1 s.

14. soal fisika latihan usbn mohon dibantu


RELATIVITAS

Soal nomor (1)

Diketahui:

Kecepatan pesawat A menurut pengamat di bumi, vap = 0,5c

Kecepatan pesawat B menurut pengamat di bumi, vbp = -0,4c (arah berlawanan dengan pesawat A)

Ditanya:

a. Kecepatan pesawat A menurut pilot pesawat B, vab = ...?

b. Kecepatan pesawat B menurut pilot pesawat A, vba = ...?

Pembahasan:

a. Kecepatan pesawat A menurut pilot pesawat B

vab = (vap + vpb)/(1+vap.vpb/c²)

vab = (0,5c + 0,4c)/(1+0,5c.0,4c/c²)

vab = 0,9c/1,2

vab = 3c/4

b. Kecepatan pesawat B menurut pilot pesawat A

vba = (vbp + vpa)/(1+vbp.vpa/c²)

vab = [-0,4c + (-0,5c)]/[1+(-0,4c).(-0,5c)/c²]

vab = -0,9c/1,2

vab = -3c/4

Soal nomor (2)

Diketahui:

Usia awal A, ta1 = 10 tahun

Usia awal B, tb1 = 10 tahun

Usia akhir A, ta2 = 40 tahun

Ditanya:

Usia B menurut A, tb' = ...?

Pembahasan

Selisih usia B menurut pengamat di bumi

Δtb = 40 - 10 = 30 tahun

Selisih usia B menurut A

Δtb' = Δtb/√(1 - v²/c²)

Δtb' = 30/√[1 - (0,8c)²/c²]

Δtb' = 30/√(1 - 0,64)

Δtb' = 30/√0,36

Δtb' = 30/0,6

Δtb' = 50

tb' = 10 + 50 = 60 tahun

Jadi usia B menurut A adalah 60 tahun

Soal nomor (3)

Diketahui:

Kecepatan pesawat 1 menurut pengamat di bumi, v13 = 0,5c

Kecepatan pesawat 2 menurut pengamat di bumi, v23 = -0,25c (arah berlawanan dengan pesawat 1)

Ditanya:

a. Kecepatan pesawat 1 menurut pilot pesawat 2, v12 = ...?

b. Kecepatan pesawat 2 menurut pilot pesawat 1, v21 = ...?

Pembahasan:

a. Kecepatan pesawat 1 menurut pilot pesawat 2

v12 = (v13 + v32)/(1+v13.v32/c²)

v12 = (0,5c + 0,25c)/(1+0,5c.0,25c/c²)

v12 = 0,75c/1,125

v12 = c/15

b. Kecepatan pesawat 2 menurut pilot pesawat 1

v21 = (v23 + v31)/(1+v23.v31/c²)

v21 = [-0,25c + (-0,5c)]/[1+(-0,25c).(-0,5c)/c²]

v21 = -0,75c/1,125

v21 = -c/15

Soal nomor (4)

Diketahui:

Panjang batang diamati saat diam, Lo = 1 m

Panjang batang diamati saat bergerak, L = 0,8 m

Ditanya:

Hitung kecepatan gerak batang, v = ...?

Pembahasan:

L = Lo.√(1 - v²/c²)

0,8 = 1.√(1 - v²/c²)

0,8 = √(1 - v²/c²)

0,64 = 1 - v²/c²

v²/c² = 1 - 0,64

v²/c² = 0,36

v² = 0,36c²

v = 0,6c
