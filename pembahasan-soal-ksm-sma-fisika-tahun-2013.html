17. tolong bantuin soal un tahun 2013


Ingat rumus:
Lim x → ∞  (√(ax² + bx + c) - √(px²  + qx + r))
= (b - q)/(2√a) ----->> (Jika a = p)
= ∞ ------------------->> (Jika a > p)
= -∞ ------------------>> (Jika a < p)

Penyelesaian:
Lim x → ∞     (3x - 1 - √(9x²  + 3x - 5))
= Lim x → ∞  (√(3x - 1)² - √(9x²  + 3x - 5))
= Lim x → ∞  (√(9x² - 6x + 1) - √(9x²  + 3x - 5))
= (b - q)/(2√a)
= (-6 - 3)/(2√9)
= -9/(2.3)
= -9/6
= -3/2

29. tolong yg jago KSM pliss buat belajar soalnya​


Diketahui :m = banyak huruf idgham bighunnahn = banyak huruf idzhar

Ditanya :Angka satuan dari m²⁰¹⁹ x n¹⁰⁰⁰ = ?

Penyelesaian :

Bagian 1 : Menentukan nilai dari m dan n

Huruf idgham bighunnah adalah ya', nun, mim, dan wawu. Sehingga, banyaknya huruf idgham bighunnah adalah 4.

Jadi, m = 4

Huruf idzhar adalah ghoin, 'ain, kho', kha', ha', dan hamzah. Sehingga, banyaknya huruf idgham bighunnah adalah 6

Jadi, n = 6

Bagian 2 : Menentukan angka satuan tiap angka

4¹ = 4

4² = 16

4³ = 64 -> Angka satuan terulang

...

4²⁰¹⁹ = 2019 / 2 = 1009 sisa 1

Sehingga, angka satuan dari 4²⁰¹⁹ adalah 4

6¹ = 6

6² = 36 -> Angka satuan sama

Karena angka satuan sama, maka angka satuan dari 6¹⁰⁰⁰ adalah 6

Sehingga,

4²⁰¹⁹ x 6¹⁰⁰⁰

= 4 x 6

= 24 (diambil angka 4-nya)

Kesimpulan :

Angka satuan dari m²⁰¹⁹ x n¹⁰⁰⁰ adalah 4 (B)


30. Komposisi massa emas di dalam logam campuran jika di ukur di udara [soal ksm fisika nomor.18 kab/kota tahun 2021]


Komposisi massa emas di dalam logam campuran tidak berubah jika diukur di udara. Udara tidak memiliki pengaruh langsung terhadap komposisi massa emas di dalam logam campuran. Pengukuran komposisi massa emas dalam logam campuran dilakukan berdasarkan proporsi relatif massa emas terhadap massa total campuran tersebut, yang tidak dipengaruhi oleh keberadaan udara.

32. Soal Latihan USBN 2017 / 2018 Mapel: Matematika (Peminatan) Satuan Pendidikan: SMA Tahun: 2017 / 2018 Kurikulum: Kurikulum 2013


Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5
