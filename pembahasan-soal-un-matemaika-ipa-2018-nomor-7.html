total = 80 siswa
matematika = 45 siswa
bahasa inggris = 40 siswa

ipa + bahasa inggris = 12 siswa

jumlah siswa yang senang matematika dan ipa saja = 18 siswa - 4 siswa = 14 siswa

16. Need help matemaika Diketahui sistem persamaan linear : 1/x + 1/y = 2 2/y - 1/z = -3 1/x - 1/z = 2 nilai x+y+z adalah....


1/x = A
1/y = B
1/z = C

1/x + 1/y = 2
A + B = 2
B = 2 - A

1/x - 1/z = 2
A - C = 2
C = A - 2

2/y - 1/z = -3
2B - C = -3
2(2 - A) - (A - 2) = -3
4 - 2A - A + 2 = -3
-3A = -9
A = 3

B = 2 - A = 2 - 3 = -1
C = A - 2 = 3 - 2 = 1

A = 3
1/x = 3
x = 1/3

B = -1
1/y = -1
y = -1

C = 1
1/z = 1
z = 1

x + y + z = 1/3 - 1 + 1 = 1/3
