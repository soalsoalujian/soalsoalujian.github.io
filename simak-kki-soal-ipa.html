1. Ini gimana ya cara-nya dari simak kki makasih..


Jika [tex]\sqrt[3]{x+7}-\sqrt[3]{x-7}=2[/tex] maka x² berada di antara (D). 45 dan 55.

PEMBAHASAN

DIKETAHUI

[tex]\sqrt[3]{x+7}-\sqrt[3]{x-7}=2[/tex]

.

DITANYA

Tentukan nilai x² berada pada range berapa.

.

PENYELESAIAN

[tex]misal:\\\\a=\sqrt[3]{x+7}\\\\a^3=x+7\\\\x=a^3-7\\\\\\\sqrt[3]{x+7}-\sqrt[3]{x-7}=2\\\\a-\sqrt[3]{a^3-7-7}=2\\\\a-2=\sqrt[3]{a^3-14}~~~~~~~...pangkat~3~kan~kedua~ruas\\\\(a-2)^3=(\sqrt[3]{a^3-14})^3\\\\a^3-6a^2+12a-8=a^3-14\\\\-6a^2+12a+6=0\\\\a^2-2a-1=0\\\\a^2-2a=1\\\\a^2-2a+1=1+1\\\\(a-1)^2=2\\\\a-1=\pm\sqrt{2}\\\\a=1\pm\sqrt{2}\\[/tex]

.

[tex]Untuk~a=1+\sqrt{2}~:\\\\x=(1+\sqrt{2})^3-7\\\\x=1+3(1)^2(\sqrt{2})+3(1)(\sqrt{2})^2+(\sqrt{2})^3-7\\\\x=5\sqrt{2}\\\\\\Untuk~a=1-\sqrt{2}~:\\\\x=(1-\sqrt{2})^3-7\\\\x=1+3(1)^2(-\sqrt{2})+3(1)(-\sqrt{2})^2+(-\sqrt{2})^3-7\\\\x=-5\sqrt{2}\\[/tex]

.

Maka :

[tex]x^2=(\pm5\sqrt{2})^2\\\\x^2=50\\[/tex]

.

KESIMPULAN

Jika [tex]\sqrt[3]{x+7}-\sqrt[3]{x-7}=2[/tex] maka x² berada di antara (D). 45 dan 55.

.

Jawaban:

-b + 3a ≤ x ≤ b + a ( E )

Penjelasan:

[x - 2a] + a ≤ b

[x - 2a] ≤ b - a

Teori umum :

[a] < b maka

-b < a < b sehingga :

[x - 2a] ≤ b - a

-(b-a) ≤ x - 2a ≤ b - a

-b + a ≤ x - 2a ≤ b - a         ( ketiga ruas + 2a)

-b + 3a ≤ x ≤ b + a ( E )

JIKA b > a, NILAI x YANG MEMENUHI | x - 2a | + a ≤ b adalah E. -b + 3a ≤ x ≤ b + a

PEMBAHASAN :

•menggunakan cara positif =

x - 2a + a ≤ b

x - 2a ≤ b - a

x ≤ b + a

•menggunakan cara negatif =

- x + 2a + a ≤ b

- x ≤ b - a - 2a

- x ≤ b - 3a

x ≥ -b + 3a

atau sama juga dengan -b + 3a ≤ x ≤ b + a

26. Bagaimana caranya ini soal simak ui juga 2016​


Jawab:

36

Penjelasan dengan langkah-langkah:

[tex]x^2+ax+b=0[/tex]

Akar"nya

[tex]x_1+x_2=-a\\x_1.x_2=b[/tex]

[tex]x^2+cx+a=0[/tex]

Akar"nya

[tex]x_3+x_4=-c\\x_3.x_4=a[/tex]

Diketahui akar [tex]x_1,x_2[/tex] adalah [tex]\frac{1}{3}[/tex] kali akar [tex]x_3,x_4[/tex]

[tex]x_1=\frac{1}{3} (x_3)[/tex] dan [tex]x_2=\frac{1}{3} (x_4)[/tex]

Sehingga

[tex]x_1+x_2=\frac{1}{3} (x_3)+\frac{1}{3} (x_4)\\x_1+x_2=\frac{1}{3} (x_3+x_4)\\-a=\frac{1}{3} (-c)\\3a=c[/tex]

dan

[tex]x_1.x_2=\frac{1}{3} (x_3).\frac{1}{3} (x_4)\\x_1.x_2=\frac{1}{9}(x_3.x_4)\\b=\frac{1}{9}a\\9b=a[/tex]

Maka;

[tex]\frac{a+c}{b} = \frac{9b+3a}{b} = \frac{9b+3(9b)}{b}= \frac{9b+27b}{b}=\frac{36b}{b}=36[/tex]

33. [Soal Latihan SIMAK]​


Jawaban:

2 dan 4 (C)

Penjelasan:

Operasi Perkalian Matriks

[tex]A=\left[\begin{array}{cc}a&b\\c&d\end{array}\right]\, \ B=\left[\begin{array}{cc}e&f\\g&h\end{array}\right][/tex]

Maka

[tex]A.B=\left[\begin{array}{cc}a&b\\c&d\end{array}\right].\left[\begin{array}{cc}e&f\\g&h\end{array}\right]=\left[\begin{array}{cc}ae+bg&af+bh\\ce+dg&cf+ch\end{array}\right][/tex]

Sehingga

[tex]\left[\begin{array}{cc}\tan x&1\\1&\tan x\end{array}\right].\left[\begin{array}{c}\cos^2 x\\\sin x \cos x\end{array}\right]=\left[\begin{array}{c}a\\b\end{array}\right]\frac{1}{2}[/tex]

Karena b=2a maka

[tex]\left[\begin{array}{cc}\tan x&1\\1&\tan x\end{array}\right].\left[\begin{array}{c}\cos^2 x\\\sin x \cos x\end{array}\right]=\left[\begin{array}{c}a\\2a\end{array}\right]\frac{1}{2}\\\\\left[\begin{array}{c}\tan x \cos^2 x+\sin x\cos x\\\cos^2 x+\tan x\sin x \cos x\end{array}\right]=\left[\begin{array}{c}\frac{1}{2}a\\a\end{array}\right][/tex]

[tex]\left[\begin{array}{c}\frac{\sin x}{\ cos x} \cos^2 x+\sin x\cos x\\\cos^2 x+\frac{\sin x}{\ cosx}\sin x \cos x\end{array}\right]=\left[\begin{array}{c}\frac{1}{2}a\\a\end{array}\right]\\\\\left[\begin{array}{c}\sin x \cos x+\sin x\cos x\\\cos^2 x+\sin^2 x\x\end{array}\right]=\left[\begin{array}{c}\frac{1}{2}a\\a\end{array}\right]\\\\\left[\begin{array}{c}2\sin x \cos x\\\cos^2 x+\sin^2 x\x\end{array}\right]=\left[\begin{array}{c}\frac{1}{2}a\\a\end{array}\right][/tex]

[tex]\left[\begin{array}{c}\sin 2x\\\cos^2 x+\sin^2 x\x\end{array}\right]=\left[\begin{array}{c}\frac{1}{2}a\\a\end{array}\right][/tex]

Didapat 2 persamaan :

[tex]\sin 2x=\frac{1}{2}a\\\sin^2 x+\cos^2 x=a[/tex]

Oleh karenanya perlu mencari nilai [tex]a[/tex] terlebih dahulu

[tex]\sin^2 x+\cos^2 x=a\\1=a\\a=1[/tex]

Substitusi nilai [tex]a=1[/tex]

[tex]\sin 2x=\frac{1}{2}a\\\sin 2x=\frac{1}{2}[/tex]

Sehingga didapat persamaan trigonometri

[tex]\sin 2x=\frac{1}{2}[/tex] untuk [tex]0\leq x\leq p[/tex]  ([tex]p[/tex] dalam hal ini [tex]\pi[/tex])

Maka

[tex]\sin 2x=\sin 30[/tex]

Solusi umum persamaan trigonometrinya

[tex]2x=30+k.360\\x=15+k.180[/tex]

atau

[tex]2x=(180-30)+k.360\\2x=150+k.360\\x=75+k.180[/tex]

Solusi khusus untuk himpunan nilai x nya yaitu :

1. [tex]x=15+k.180[/tex]

Untuk [tex]k=0[/tex]

[tex]x=15+0.180\\x=15 \iff x=\frac{p}{12}\ \ \ (Memenuhi)[/tex]

Untuk [tex]k=1[/tex]

[tex]x=15+1.180\\x=195 \iff x=\frac{13p}{12}\ \ \ (Tidak\ Memenuhi)[/tex]

2. [tex]x=75+k.180[/tex]

Untuk [tex]k=0[/tex]

[tex]x=75+0.180\\x=75 \iff x=\frac{5p}{12}\ \ \ (Memenuhi)[/tex]

Untuk [tex]k=1[/tex]

[tex]x=75+1.180\\x=255 \iff x=\frac{17p}{12}\ \ \ (Tidak\ Memenuhi)[/tex]

Jadi, Himpunan untuk nilai x : [tex]HP=\{x|x=\frac{p}{12} \cup x=\frac{5p}{12}\}[/tex]

Oleh karenanya jawaban yang benar ada adalah 2 dan 4 (C)

Jawaban:

KKI, atau Kesatuan Gereja-Gereja di Indonesia, adalah sebuah organisasi gerejawi yang beranggotakan gereja-gereja Kristen di Indonesia. Organisasi ini didirikan dengan tujuan untuk memperkuat kesatuan dan kerja sama antara gereja-gereja Kristen di Indonesia dalam menghadapi berbagai isu dan tantangan.

Perkembangan KKI dimulai pada tahun 1945, ketika Indonesia meraih kemerdekaannya dari penjajahan Belanda. Pada masa itu, terdapat berbagai denominasi gereja Kristen yang beroperasi di Indonesia dengan latar belakang budaya, teologi, dan struktur organisasi yang berbeda. Melalui berbagai dialog dan pertemuan, gereja-gereja tersebut menyadari pentingnya kesatuan dalam menghadapi masa depan yang baru.

Pada tanggal 9 Februari 1950, KKI secara resmi didirikan di Malino, Sulawesi Selatan. Pendirian ini menjadi tonggak penting dalam upaya menyatukan gereja-gereja Kristen di Indonesia. KKI mendasarkan dirinya pada visi dan misi untuk mempromosikan persekutuan, keadilan sosial, perdamaian, dan kasih di tengah-tengah masyarakat Indonesia.

Seiring berjalannya waktu, KKI terus berkembang dan bekerja sama dengan gereja-gereja anggota untuk mengatasi berbagai tantangan dan isu-isu sosial, ekonomi, dan politik yang dihadapi oleh bangsa Indonesia. Organisasi ini juga aktif dalam bidang pendidikan, pelayanan sosial, dan advokasi hak asasi manusia.

Selama perjalanan sejarahnya, KKI tetap konsisten dalam memperjuangkan toleransi, kerukunan antaragama, dan penguatan nilai-nilai keadilan dan perdamaian di Indonesia. Dengan mempertahankan semangat kebersamaan dan kerjasama antargereja, KKI terus berkontribusi dalam memajukan bangsa dan negara Indonesia serta memperkuat peran gereja-gereja Kristen dalam menyebarkan ajaran kasih dan pengharapan bagi seluruh masyarakat.
