= lim x-->0 √x(√(1+x) - 1) / x√x
= lim x -->0 (√(1+x) - 1)/x
= lim x -->0 (√(1+x) - 1)/x * (√(1+x) +1)/(√(1+x) +1)
= lim x--> 0 1+x-1 / (x * (√(1+x) +1))
= lim x-->0 x /( x * (√(1+x) +1))
= lim x--> 0 1/(√(1+x) +1)
= 1/(√1 +1) = 1/2 (D)Lim (√(x + x^2) - √x) / x√x . (√(x + x^2) + √x)/(√(x + x^2) + √x)

12. Contoh soal dan pembahasan limit kelas 10


Jika f(x) = x2 − 6x + 8, tentukan interval f(x) naik dan interval f(x) turun!

Jawab :
f '(x) = 2x − 6

f(x) naik ⇒ f '(x) > 0
⇔ 2x − 6 > 0
⇔ 2x > 6
⇔ x > 3

f(x) turun ⇒ f '(x) < 0
⇔ 2x − 6 < 0
⇔ 2x < 6
⇔ x < 3

Jadi f(x) naik pada interval x > 3 dan turun pada interval x < 3.

a) f(x) = 3x + 2x − 5x
f ‘(x) = 4⋅3x + 2⋅2x − 5x^1-1

f ‘(x) = 12x + 4x − 5

36. matematika bantu menjawab soal limit​


>>Limit Aljabar<<

Diketahui:

[tex]\lim_{x \to 3} (\frac{x^2+3x-18}{x^2-3x} )[/tex]

Maka, [tex]\lim_{x \to 3} (\frac{x^2+3x-18}{x^2-3x} ) =\bold{\underline{3}} \checkmark \checkmark[/tex]
