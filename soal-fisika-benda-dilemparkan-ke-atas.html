Pembahasan :

Dari uraian soal 1 - 5 merupakan gerak vertikal. Gunakan konsep energi mekanik, dimana energi mekanik awal = energi mekanik akhir. Perumusan sebagai berikut

                                 [tex]\rm EM_1= EM_2\\\rm EP_1 +EK_1 = EP_2 +EK_2\\\rm mgh_1 + \dfrac{1}{2}mv_1^2 = mgh_2 + \dfrac{1}{2}mv_2^2[/tex]

Keterangan :

EM = energi mekanik

EP = energi potensial

EK = energi kinetik

↓

1. Vertikal ke atas

Diketahui :

m = 2 kgv = 25 m/s

Ditanya :

V saat h = 20 m

Dijawab :

[tex]\rm EM_1= EM_2\\\rm EP_1 +EK_1 = EP_2 +EK_2\\\rm mgh_1 + \dfrac{1}{2}mv_1^2 = mgh_2 + \dfrac{1}{2}mv_2^2\\\rm 0 + \dfrac{1}{2}2(25)^2 = 2(10)20+\dfrac{1}{2}2v_2^2\\ \rm 625 = 400+v_2^2\\\rm v_2 = \sqrt{625-400}\\ \rm v_2 = 15 m/s[/tex]

2. Vertikal ke bawah

Diketahui :

m = 2 kgh = 35 m

Ditanya :

v saat h = 5 m

Dijawab :

[tex]\rm EM_1= EM_2\\\rm EP_1 +EK_1 = EP_2 +EK_2\\\rm mgh_1 + \dfrac{1}{2}mv_1^2 = mgh_2 + \dfrac{1}{2}mv_2^2\\\rm 2(10)(35) + 0 =2(10)(5) + \dfrac{1}{2}2v_2^2\\\rm 700 = 100+v_2^2\\\rm v_2=\sqrt{700-100} \\\rm v_2 = 24,49\:m/s[/tex]

3. Vertikal ke atas

Diketahui :

m = 3 kg[tex]\rm v_o[/tex] = 18 m/s

Ditanya :

ketinggian saat [tex]\rm v_2[/tex] 10m/s

Dijawab :

[tex]\rm EM_1= EM_2\\\rm EP_1 +EK_1 = EP_2 +EK_2\\\rm mgh_1 + \dfrac{1}{2}mv_1^2 = mgh_2 + \dfrac{1}{2}mv_2^2\\3(-10)(0) + \dfrac{1}{2}3(18)^2=3(10)h_2 +\dfrac{1}{2}3(10)^2\\486 =30h_2+150\\\rm h_2 =\dfrac{486-150}{30}\\ \rm h_2 = 11,2 \:m[/tex]

4. Vertikal ke bawah

Diketahui :

m = 2 kg[tex]\rm h_1[/tex] = 42 m

Ditanya :

[tex]\rm h_2[/tex] saat [tex]\rm v_2[/tex] = 15 m/s

Dijawab :

[tex]\rm EM_1= EM_2\\\rm EP_1 +EK_1 = EP_2 +EK_2\\\rm mgh_1 + \dfrac{1}{2}mv_1^2 = mgh_2 + \dfrac{1}{2}mv_2^2\\2(10)(42) + 0 = 2(10)h_2+ \dfrac{1}{2}2(15)^2\\840 = 20h_2+225\\\rm h_2 = \dfrac{840-225}{20}\\ \rm h_2 = 30,75\:m[/tex]

h= (vo.sinx)² / 2g
= (20.1)² / 2.10
= 400 / 20
= 20 m

Ep = m.g.h
= 1.10.20
= 200 N
dik : m = 1 kg ; Vo = 20 m/s ; α = 90
dit : Ep = ...???
jawab :
*) cari dulu tinggi maks nya :
h.maks = (Vo.sin α)^2 : 2g = (20.sin 90)^2 : 2.10 = 400 : 20 = 20 m

**) cari Ep :
Ep = m.g.h = 1.(10).(20) = 200 joule.
