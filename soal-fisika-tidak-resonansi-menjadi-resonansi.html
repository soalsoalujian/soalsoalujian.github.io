40. pada tabung resonansi terjadi resonansi kedua ketika panjang kolom udara 48 cm. panjang kolom udara ketika terjadi resonansi keempat adalah


L / L' = [1/4 λ (2n - 1)] / [1/4 λ (2n' - 1)]
L' = L (2n' - 1) / (2n - 1)
    = (48 cm) (2·4 - 1) / (2·2 - 1) = 112 cmPanjang kolom udara pada tabung,
saat resonansi ke-2, L2 = 48 cm
saat resonansi ke-4, L4 = ___?

Jawab :

Panjang kolom udara oada tabung,
saat resonansi pertama, kedua, ketiga, keempat, kelima, dan seterusnya adalah mengikuti BILANGAN GANJIL, maka 
L1 : L2 : L3 : L4 : L5 : ....... =
1   :  3  :   5 :  7  :  9 :  .......

Jadi 
L2 : L4 = 3 : 7
48 : L4 = 3 : 7

L4 = (7/3). 48

L4 = 7. 16

L4 = 112 cm
