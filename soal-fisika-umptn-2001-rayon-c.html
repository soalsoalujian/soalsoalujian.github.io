8. Soal no 510 Umptn/b/1995


f(x) = x^3 + 3x^2 + 5 akan turun jika f'(x) < 0
f'(x) = 3x^2 + 6x < 0
=> 3x(x + 2) < 0
x = 0 atau x = -2
+++ (-2) --- (0) +++
-2 < x < 0Kelas : XI (2 SMA)
Materi : Turunan
Kata Kunci : fungsi turun, penyelesaian

Pembahasan :
Diketahui fungsi kuadrat f(x) = x³ + 3x² + 5
Agar fungsi f(x) merupakan fungsi turun, maka f'(x) < 0.
Sehingga
3x² + 6x < 0

Untuk menentukan penyelesaian dari pertidaksamaan kuadrat, maka kita ubah dahulu menjadi persamaan kuadrat
3x² + 6x = 0
⇔ x² + 2x = 0
⇔ x(x + 2) = 0
⇔ x = 0 V x + 2 = 0
⇔ x = 0 V x = -2
Perhatikan gambar terlampir.
Daerah yang negatif merupakan daerah fungsi yang turun, yaitu
-2 < x < 0.

13. 8. Peluang Aswan lulus UMPTN adalah 0,95. Peluang Agus lulus UMPTN adalah 0,90. Peluang salah satu dari ke-duanya lulus adalah ........ : a. 0,855 b. 0,14 c. 0,10 d. 0,05 e. 0,005


jawab
A = Aswan
B = Agus
P(Alulus) = 0,95
P(A' tidak lulus) = 1 - 0,95 = 0,05

P(B lulus) = 0.90
P(B' tidak lulus) = 1 - 0,90 = 0,10

Peluang salah satu yang lulus, 
= P(A lulus dan B tidak lulus) atau P(A tidak lulus dan B lulus)
= { PA x PB' } + {PA' x PB}
= (0,95 x 0,10) + (0,05 x 0,90)
= 0,14

[tex]\boxed{\boxed{\boxed{\tt Q = m \times c \times \Delta T}}}\\[/tex]
