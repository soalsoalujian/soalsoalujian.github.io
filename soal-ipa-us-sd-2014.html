6. Tentukan a. banyak angka 0 pada hasil bagi 2014 2014 2014 dibagi 2014 b. apabila angka 2 1 0 dan 4 masing-masing terdapat 300 angka seperti pola soal a berapakah hasil baginya ketika dibagi 2014


a. Banyak angka 0 pada hasil bagi 2014 2014 2014 : 2014 adalah 6.

b.  Angka 2, 1, 0, dan 4 masing-masing terdapat 300 angka seperti pola soal a, maka hasil baginya ketika dibagi 2014 adalah 897.  

Penjelasan dengan langkah-langkah

Diketahui :

Bilangan 2014 2014 2014

Ditanya :

a. Banyak angka 0 hasil dari 2014 2014 2014 : 2014.

b. Angka 2, 1, 0, dan 4 masing-masing terdapat 300 angka seperti pola soal a, maka hasil baginya ketika dibagi 2014.

Jawab :

a. Menentukan banyak angka 0

Hasil bagi dari 2014 2014 2014 : 2014 = 100010001

Banyak angka 0 pada 100010001 adalah 6.

b. Menentukan banyak angka 0 pada susunan pengulangan 300 kali

Berdasarkan pola soal a bisa kita buat.

2014 2014 2014 : 2014 = 100010001 ⇒ 3 susunan, nolnya 6

2014 2014 2014 2014 : 2014 = 1000100010001 ⇒ 4 susunan, nolnya 9

2014 2014 2014 2014 : 2014  = 10001000100010001 ⇒ 5 susunan, nolnya 12.

Karena selisih banyak 0 tiap susunan ada 3 dan n = banyak susunan penulangan, maka dapat kita buat rumusnya 3 × n - 3.

Banyak 0 pada susunan 300

= 3n - 3

= 3 × 300 - 3

= 900 - 3

= 897

34. pada tahun 2014 data keuangan Filipina mencatat pendapatan domestik bruto PDB riil sebesar 272,02 miliar US Dollar pada tahun 2013 PDB Filipina sebesar 250,18 miliar US Dollar Berdasarkan data tersebut hitunglah laju pertumbuhan ekonomi Filipina tahun 2014


Diketahui:
PDB tahun 2014= US$272,02 miliar
PDB tahun 2013= US$250,18 miliar

Ditanya: Laju pertumbuhan ekonomi

Jawab:

[Rumus Laju Pertumbuhan Ekonomi]
Laju pertumbuhan ekonomi tahun t= PDBt - PDBt-1/ PDBt-1 × 100%

[Keterangan]
tahun t= tahun terhitung
PDBt= PDB tahun terhitung
PDBt-1= PDB sebelum tahun terhitung

Laju pertumbuhan ekonomi tahun 2014= PDB tahun 2014-PDB tahun 2013/ PDB tahun 2013 × 100%

Laju pertumbuhan ekonomi tahun 2014= US$272,02 miliar-US$250,18 miliar/US$272,02 miliar × 100%

Laju pertumbuhan ekonomi tahun 2014= US$21,84 miliar/272,02 miliar × 100%

Laju pertumbuhan ekonomi tahun 2014= 8,03%

Jadi, laju pertumbuhan ekonomi Filipina pada tahun 2014 adalah sebesar 8,03%.

38. Bagaimana cara mengetahui nilai UN SD tapi di ijasah cuma ada nilai US karna tahun 2014 UN SD sudah di hapus?


Untuk dapat mengetahu nilai hasil UN SD adalah dengan melihat SKHUN bukan pada ijazah.

Penjelasan:

Nilai UN atau nilai Ujian nasional merupakan indikator kelulusan pada siswa baik pada tingkat Sekolah Dasar, Sekolah Menengah Pertama ataupun Sekolah Menengah Atas. Beberapa tahun belakangan, nilai Ujian Nasional diletakkan sendiri pada lembar SKHUN atau kepanjangan dari Surat Keterangan Hasil Ujian Nasional. Jadi pada intinya nilai khusus pada mata pelajaran yang diujikan pada Ujian Nasional akan dipisahkan dari hasil nilai ijazah. Nilai yang ada pada ijazah ada;ah nilai-nilai hasil dari perhitungan dan pengolahan raport dari kelas 4 sampai dengan kelas 6 saat di bangku Sekolah Dasar. Bisa dikatakan bahwa nilai Ujian Nasional merupakan nilai tersendiri dan tidak bisa dicampurkan dengan nilai ijazah, akan tetapi nilai yang ada dala ijazah merupakan hasil pengolahan nilai raport dan nilai-nilai ujian lain termasuk juga nilai hasil Ujian Nasional. Adapun dalam pelaksanaan Ujian Nasional pada tahun 2014 mata pelajaran yang dibuat untuk pelaksanaan ujian hanyalah 3 mata pelajaran yaitu: Bahasa Indonesia, Matematika dan Ilmu Pengetahuan Alam (IPA).
