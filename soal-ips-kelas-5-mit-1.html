23. limit mit mit mit mit........................... 1. [tex] \lim_{x \to \infty} (2^x^+^1~+3^x^-^5+~4^x^+^2):(2^x-3^x^+^1+4^x^-^1)[/tex] =.....??? 2. [tex] \lim_{x \to 0} \frac{ \sqrt[3]{1+sinx}~-~ \sqrt[3]{1-sinx} }{x} =?????[/tex]keterangan no 2 = itu adalah akar pangkat 3 oke bro????makasih ya....


limit x menuju tak hingga
(2^(x+1) + 3^(x-5) + 4^(x+2)) / (2^x - 3^(x+1) + 4^(x-1))

Anggapan limit menuju tak hingga plus,
Jadi, ingat konsep limit tak hingga plus,

Setiap konstanta dibagi dengan pembilang/penyebut dengan nilai pangkat terbesar.
Abaikan 2 dan 3, kita memiliki 4

= limit x menuju tak hingga
4^(x+2) / 4^(x-1)

Bagi dengan pangkat terbesar,

= limit x menuju tak hingga
(4^(x+2) / 4^(x + 2)) / (4^(x-1) / 4^(x+2))
= 1 / 4^(x-1-x-2)
= 1 / 4^(-3)
= 4^3
= 64

==========

limit x menuju nol
∛(1+sinx) - ∛(1-sinx) / x

Turunkan

= limit x menuju nol
1/3(cosx)(1/(sinx+1)^(2/3) + 1/(sinx-1)^(2/3)) / 1

Masukan nilai nol

= 1/3(cos 0)(1/(sin 0 + 1)^(2/3) + 1/(sin 0 - 1)^(2/3)) / 1
= 1/3(1)(1/(0+1)^(2/3) + 1/(0-1)^(2/3))
= 1/3(1/1 + 1/1)
= 1/3(2)
= 2/3
