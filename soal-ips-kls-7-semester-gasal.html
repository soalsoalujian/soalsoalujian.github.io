32. Yang memenuhi pertamanan .. No 4&5 X MIPA SEMESTER GASAL


4) 4/(x - 3) ≥ 3/(x + 2)
4/(x - 3) - 3/(x + 2) ≥ 0
(4(x + 2) - 3(x - 3))/(x - 3)(x + 2) ≥ 0
(4x + 8 - 3x + 9)/(x - 3)(x + 2) ≥ 0
(x + 17)/(x - 3)(x + 2) ≥ 0
x = -17 atau x = 3 atau x = -2
Garis bilangan
---- [-17] +++ (-2) --- (3) +++
-17 ≤ x < -2 atau x > 3

5) 2/(x + 4) < 5/(x - 2)
2/(x + 4) - 5/(x - 2) < 0
(2(x - 2) - 5(x + 4))/(x + 4)(x - 2) < 0
(2x - 4 - 5x - 20)/(x + 4)(x - 2) < 0
(-3x - 24)/(x + 4)(x - 2) < 0
-3(x + 8)/(x + 4)(x - 2) < 0
x = -8 atau x = -4 atau x = 2
Garis bilangan
+++ (-8) --- (-4) +++ (2) ---
-8 < x < -4 atau x > 2
