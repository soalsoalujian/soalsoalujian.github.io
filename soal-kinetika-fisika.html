5. sebutkan orde reaksi kinetika kimia


Orde Satu :  t1/2 =  ln 2 / k  =  0,693 / k   (waktu paruh tidak bergantung pada konsentrasi awal reaktan)

Orde Dua :   t1/2 =  1 / k.[A]0 (waktu paruh berbanding terbalik dengan konsentarsi awal reaktan)

Orde Nol :   t1/2 =  [A]0 / 2k   (waktu paruh berbanding lurus dengan konsentrasi awal reaktan)

37. 2. Suatu larutan dengan kinetika orde kedua memiliki waktu paruh 50 menit dan konstanta laju reaksi 12 L/mol. Hitung berapa konsentrasi awal zat tersebut dan t90 nya!​


Jawaban:

**Konsentrasi awal zat**

Waktu paruh orde kedua adalah waktu yang dibutuhkan untuk konsentrasi zat berkurang menjadi setengah dari konsentrasi awal.

```

[A]t = [A]0 / 2

```

dimana:

* [A]t adalah konsentrasi zat pada waktu t

* [A]0 adalah konsentrasi zat awal

Dengan menggunakan nilai waktu paruh dan konstanta laju reaksi, kita dapat menghitung konsentrasi awal zat sebagai berikut:

```

[A]0 = [A]t * 2 / k

```

```

[A]0 = ([A]0 / 2) * 2 / 12 L/mol

```

```

[A]0 = 12 L/mol

```

Jadi, konsentrasi awal zat tersebut adalah **12 L/mol**.

**Waktu 90%**

Waktu 90% adalah waktu yang dibutuhkan untuk konsentrasi zat berkurang menjadi 10% dari konsentrasi awal.

```

[A]t = [A]0 / 10

```

Dengan menggunakan nilai konsentrasi awal dan konstanta laju reaksi, kita dapat menghitung waktu 90% sebagai berikut:

```

t90 = ln(0.1) / k

```

```

t90 = ln(0.1) / 12 L/mol

```

```

t90 = 3.32 jam

```

Jadi, waktu 90% zat tersebut adalah **3.32 jam**.

Jawaban:

* Konsentrasi awal zat: 12 L/mol

* Waktu 90% zat: 3.32 jam
