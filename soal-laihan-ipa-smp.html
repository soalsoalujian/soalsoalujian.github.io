22. SOAL LAIHANLimit Fungsi Aljabar tak Hingga Kerjakan pada kertas 1 lembar1. Tentukanlah nilai dari lim x∞8x3+ 7x² + 12x3+2x-1=......2. Tentukan nilai limit dari √4x + 6-√4x4 =........​


Jawaban

1.) = 4

2.) = 0

Pembahasan

.

Nomor 1

Faktorkan dengan pangkat tertinggi

[tex] = \lim _{x \to \infty } \frac{8 {x}^{3} + 7 {x}^{2} + 1 }{2 {x}^{3} + 2x - 1} \\ [/tex]

[tex] = \lim_{x \to \infty } \frac{ \cancel{{x}^{3}}(8 + \frac{7}{x} + \frac{1}{ {x}^{3} }) }{ \cancel{{x}^{3}}(2 + \frac{2}{x} - \frac{1}{ {x}^{3} } ) } \\ [/tex]

[tex] = \lim _{x \to \infty } \frac{8 + \frac{7}{x} + \frac{1}{ {x}^{3} } }{2 + \frac{2}{x} - \frac{1}{ {x}^{3} } } \\ [/tex]

[tex] = \frac{ 8 + 0 + 0}{2 + 0 - 0} \\ [/tex]

[tex] = \frac{8}{2} \\ [/tex]

[tex] = 4[/tex]

.

.

[tex] = \lim _{x \to \infty } \sqrt{4x + 6} - \sqrt{4x + 4} \times \frac{ \sqrt{4x + 6} + \sqrt{4x + 4} }{ \sqrt{4x + 6} + \sqrt{4x + 4} } \\ [/tex]

[tex] = \lim _{x \to \infty } \frac{4x + 6 - 4x - 4}{ \sqrt{4x + 6} + \sqrt{4x + 4} } \\ [/tex]

[tex] = \lim _{x \to \infty } \frac{2}{ \sqrt{4x + 6} + \sqrt{4x + 4} } \\ [/tex]

[tex] = \lim _{x \to \infty } \frac{ \frac{2}{ \sqrt{x} } }{ \sqrt{4 + \frac{6}{x} } + \sqrt{4 + \frac{4}{x} } } \\ [/tex]

[tex] = \frac{0}{ \sqrt{4 + 0} + \sqrt{4 + 0} } \\ [/tex]

[tex] = \frac{0}{2 + 2} \\ [/tex]

[tex] = \frac{0}{4} \\ [/tex]

[tex] = 0[/tex]

.

.
