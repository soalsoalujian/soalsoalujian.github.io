1. sbmptn / utbk harus dilakukan semua murid di Indonesia tidak?siapa saja yang harus melakukan sbmptn / utbk?untuk apa sbmptn / utbk?​


Sbmptn/utbk tidak harus dilakukan oleh semua siswa(i) Indonesia. Sbmptn hanya diikuti oleh siswa(i) yg berada di jenjang SMA dan gagal di Snmptn, Snmpn, dan Span-ptkin atau diikuti oleh siswa(i) Indonesia yang berada di jenjang SMA tetapi tidak masuk kuota siswa eligible. Seperti namanya, Sbmptn (Seleksi bersama masuk perguruan tinggi negeri) maka sbmptn adalah tes atau seleksi untuk masuk ke perguruan tinggi negeri.

3. Apa itu UTBK dan SBMPTN?​


Jawaban:

UTBK (Ujian Tulis Berbasis Komputer) merupakan tes masuk ke perguruan tinggi yang dilaksanakan oleh Lembaga Tes Masuk Perguruan Tinggi (LTMPT) sebagai satu satunya lembaga penyelenggara tes perguruan tinggi terstandar di Indonesia.

SBMPTN (Seleksi Bersama Perguruan Tinggi Negeri) merupakan seleksi bersama dalam penerimaan mahasiswa baru di lingkungan perguruan tinggi negeri menggunakan pola uji tertulis secara nasional.

12. Soal Matematika ipa sbmptn


(a - |a - 2|)/a > 2
(a - |a - 2|)/a - 2 > 0
(a - |a - 2|) - (2a))/a > 0
(-a - |a - 2|)/a > 0

tentukan batas-batasnya terlebih dahulu.

untuk a - 2 ≥ 0, a ≥ 2

(-a - (a - 2))/a > 0
(-2a + 2)/a > 0

-2a + 2 = 0
a = 1

Uji pada titik a = 3, maka

(-2(a) + 2)/a = (-2(3) + 2)/3 = (-6 + 2)/3 = -4/3 < 0
Jadi, tidak ada bilangan yang memenuhi

untuk a - 2 < 0, a < 2

(-a - (-(a - 2))/a > 0
(a - (-a + 2))/a > 0
(a + a - 2)/a > 0
(2a - 2)/a > 0

2a - 2 = 0
a = 1
a ≠ 0 (syarat penyebut)

Uji pada titik a = -1, a = 1/2

(2(-1) - 2)/(-1) = (-2 - 2)/(-1) = 4 > 0 (memenuhi, jadi a < -1)
(2(1/2) - 2)/(1/2) = (1 - 2)/(1/2) = -2 < 0 (tidak memenuhi)

Jadi, penyelesaiannya a = {-9, -8, -7, -6, -5, -4, -3, -2, -1}

-9 - 8 - 7 - 6 - 5 - 4 - 3 - 2 - 1 = -45

22. matriks soal latihan sbmptn


det(AC) = det(A)*det(C) = 0

Karena C memiliki invers, maka det(C) ≠ 0, maka det(A) haruslah 0

sehingga,

x^2 - 2 = 0
x = ±√2

juga det(B) = 0

sehingga,

(-1) - (-y)(3) = 0
-1 + 3y = 0
y = 1/3

sehingga,

2x^2 + 27y^3 = 2(√2)^2 + 27(1/3)^3 = 4 + 1 = 5
