11. Ip address 126.46.5.6 termasuk dalam ip address kelas….


Jawaban:

IP address kelas A

Penjelasan:

IP ADDRESS

IP Address atau Alamat IP adalah alamat yang menjadi tanda pengenal untuk setiap host yang terhubung ke jaringan dengan TCP/IP (internet), berdasarkan aturan dari Internet Protocol (IP). Setiap host yang akan terhubung ke jaringan yang berbasis TCP/IP, harus memiliki IP address. IP Address bersifat unik, artinya dalam satu jaringan tidak ada dua host atau lebih yang menggunakan alamat IP yang sama.

Kelas di IP addres terdiri dari beberapa kelas yaitu

Kelas A : 1-126Kelas B : 128-191Kelas C : 192-223Kelas D : 224-239Kelas E : 240-247

Jadi IP address 126.46.5.6 termasuk ke dalam IP address kelas A

19. IP Address 126.46.5.6 termasukdalam IP Address kelas ...​


IP Address 126.46.5.6 telah termasuk ke dalam IP Address kelas A. Hal tersebut dikarenakan kelas A ini telah biasa digunakan untuk jaringan dengan mempunyai skala yang besar.

Bits yang pertama pada IP Address pada kelas A telah selalu diset dengan nilai nol (0), bits pertama tersebut telah berada pada range 1 hingga 126 di dalam bilangan yang desimal. 24 Bit sisanya tersebut telah merepresentasikan host identifier. Dengan jumlah host identifier yaitu sampai dengan 25 bits maka dari itu kelas A ini telah mempunyai 16,777,214 host.

Pembahasan

Pengertian dari IP address yaitu ialah sebaris angka yang di mana telah mempunyai setiap komputer, gawai, ataupun ponsel yang telah terhubung dengan adanya internet. Angka tersebut telah berbeda dengan tiap-tiap perangkat dan juga telah digunakan untuk menghubungi satu sama lain.

Maka dari itu, masing-masing website telah mempunyai IP addressnya sendiri seperti 74.125.224.72 yang telah digunakan oleh mesin pencarian Google. Namun, alamat IP yang dimiliki dari situs seseorang mungkin sama dengan yang telah dimiliki oleh webmaster yang lainnya.

IP Address ini ialah identitas yang dimiliki oleh suatu komputer di dalam jaringan internet. Dengan begitu, pemilik dari suatu website tersebut telah dapat mengetahui seluruh IP Address yang telah mengakses situsnya tersebut. Hal tersebut telah berlaku pula pada jaringan WIFI publik.

20. ip address 126.46.5.6 termasuk dalam ip address kelas ….


IP Address terdiri atas 32 bit (biary digit atau bilangan duaan) angka biner yang dibagi dalam 4 oket (byte) terdiri dari 8 bit. Setiap bit mempresentasikan bilangan desimal mulai dari 0 sampai 255.

Maka jawaban untuk soal diatas adalah ip address kelas A

Pembahasan:

IP Address merupakan serangkaian angka yang menjadi identitas perangkat dan terhubung ke internet atau infrastruktur jaringan lainnya. Apa fungsinya? Fungsinya bisa diumpamakan sebagai nomor rumah pada alamat, yakni memastikan agar data dikirimkan ke perangkat yang tepat. Untuk panjangnya rangkaian angka dimulai dari 0.0.0.0 hingga 255.255.255.255.

IP Address adalah alamat yang diberikan ke jaringan dan peralatan jaringan yang menggunakan protokol TCP/IP. IP Address terdiri atas 32 bit (biary digit atau bilangan duaan) angka biner yang dibagi dalam 4 oket (byte) terdiri dari 8 bit. Setiap bit mempresentasikan bilangan desimal mulai dari 0 sampai 255.

Jenis-jenis IP Address terdiri dari :

IP Public

Public bit tertinggi range address bit network address

kelas A 0 0 – 127* 8

kelas B 10 128 – 191 16

kelas C 110 192 – 223 24

kelas D 1110 224 – 239 28

Privat

IP Privat ini dapat digunakan dengan bebas tetapi tidak dikenal pada jaringan internet global. Karena itu biasa dipergunakan pada jaringan tertutup yang tidak terhubung ke internet, misalnya jaringan komputer ATM.

10.0.0.0 – 10.255.255.255

10.0.0.0 – 10.255.255.255172.16.0.0 – 172.31.255.255

10.0.0.0 – 10.255.255.255172.16.0.0 – 172.31.255.255192.168.0.0 – 192.168.255.255

Kesimpulan

1.0.0.0 - 126.0.0.0 : Kelas A.

127.0.0.0 : Loopback network.

128.0.0.0 - 191.255.0.0 : Kelas B.

192.0.0.0 - 223.255.255.0 : Kelas C.

224.0.0.0 = 240.0.0.0 : Class E, reserved.
