Dari PQRS itu C tapi dari QRS itu A

Penjelasan:

Dik :

Ces = 2100 J/kg°C

Cair = 4200 J/kg°C

M = 100 g » kg » 0,1 kg

L = 336000 J/kg

∆T1 = 5°C

∆T2 = 10°C

Dit :

Stot Q-R-S = ?

PQRS

Stot = S1 + S2 + S3

m x c x t + L x m + m x c x t

0,1 x 2100 x 5 + 336000 x 0,1 + 0,1 x 4200 x 10

= 1050 + 33600 + 4200

= 38850 J

QRS

Stot = L x m + m x c x t

336000 x 0,1 + 0,1 x 4200 x 10

= 33600 + 4200

= 37800

a = U1 = 1
b = U2 - U1 → 2

Un = a + (n-1)b
99 = 1 + (n-1)2
99 = 1 + 2n - 2
99 = 2n - 1
2n = 100
n = 100/2
n = 50

Sn = (n/2)(a + Un)
S50 = (50/2)(a + U50)
S50 = 25(1+99)
S50 = 25(100)
S50 = 2500
