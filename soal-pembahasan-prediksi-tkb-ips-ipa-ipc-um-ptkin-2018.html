1. Ilmu Perpustakaan itu termasuk IPA, IPS, atau IPC? Untuk SBMPTN-nya


Ilmu Perpustakaan itu termasuk IPA, IPS, atau IPC? Untuk SBMPTN-nya  jawabannya adalah IPS dan IPC.

Pembahasan

Ilmu perpustakaan merupakan rumpun soshum atau sosial humaniora. Rumpun ini termasuk di dalam fakultas ilmu budaya dan fakultas ilmu komunikasi. Di dalam SBMPTN, ilmu perpustakaan masuk ke dalam ujian IPS dan IPC. Adapun IPA merupakan bukan dari rumpun ilmu alam, karena ilmu perpustakaan berkaitan dengan perpustakaan dan pembukuaan buku itu sendiri.

22. Jumlah biaya tiga pendaftaran untuk sbmptn(ipa,ips,ipc) di univ A adalah 475.000. Biaya masuk ipa lebih besar 25.000 dari ips. Biaya masuk ipc lebih besar 50.000 dari biaya pendaftaran ipa. Berapa selisih dua kali biaya terbesar dengan 3 kali yang terkecil? A. 200.000 B. 25.000 C. 150.000 Jwb beserta carany ^^


Misalkan
a = IPA
b = IPS
c = IPC

maka :
a = b + 25.000
c = a + 50.000 → c = b + 25.000 + 50.000 → c = b + 75.000

a + b + c = 475.000
b + 25.000 + b + b + 75.000 = 475.000
3b + 100.000 = 475.000
3b = 475.000 - 100.000
3b = 375.000
b = 375.000 / 3

b = 125.000

a = b + 25.000
a = 125.000 + 25.000
a = 150.000

c = b + 75.000
c = 125.000 + 75.000
c = 200.000

yg terbesar adalah c dengan 200.000
yg terkecil adalah b dengan 125.000

2c - 3b = 2 (200.000) - 3(125.000) = 400.000 - 375.000 = 25.000

Jawaban B


Misalkan: ipa = x; ips = y; dan ipc = z

x + y + z = 475.000 ...persamaan 1
x = 25.000 + y → x - y = 25.000 ...persamaan 2
z = 50.000 + x → -x + z = 50.000 ...persamaan 3

Eliminasi persamaan 1 dan persamaan 2
x + y + z = 475.000
x - y = 25.000
________________+
2x + z = 500.000 ...persamaan 4

Eliminasi persamaan 3 dan persamaan 4
-x + z = 50.000
2x + z = 500.000
_______________-
-3x = -450.000
x = (-450.000) ÷ (-3) = 150.000

x - y = 25.000
150.000 - y = 25.000
-y = 25.000 - 150.000
-y = -125.000
y = 125.000

-x + z = 50.000
-150.000 + z = 50.000
z = 50.000 + 150.000
z = 200.000

Biaya terbesar (z): ipc = Rp 200.000,-
Biaya terkecil (y): ips = Rp 125.000,-

2z - 3y = 2 × 200.000 - 3 × 125.000
= 400.000 - 375.000
= Rp 25.000,- (B)
