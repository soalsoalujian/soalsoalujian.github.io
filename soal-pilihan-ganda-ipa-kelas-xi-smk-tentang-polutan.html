18. tolong di bantu yah, soal matematika SMK kelas XI ​


a. Persamaan lingkaran berpusat (-2,5), melalui (-1,2)

(x + 2)² + (y - 5)² = r²

→ (-1 + 2)² + (2 - 5)² = 10

→ x² + y² + 4x - 10y + 4 + 25 - 10 = 0

→ x² + y² + 4x - 10y + 19 = 0

b. Persamaan lingkaran berpusat (-2,5) menyinggung garis x - 3 = 0 → x = 3

(x + 2)² + (y - 5)² = r²

→ (3 + 2)² + (0 - 5)² = r²

→ r² = 25

→ (x + 2)² + (y - 5)² = 25

→ x² + y² + 4x - 10y + 4 + 25 - 25 = 0

→ x² + y² + 4x - 10y + 4 = 0

rata2 hasil ulangan mat kelas XI IPA 3 adalah 76jawab :
72=(30*72+25*68+20*n)/30+25+20
72=(2160+1700+20n)/75
72*75=3860+20n
5400-3860=20n
1540=20n
77=n
jadi, rata2 hasil ulangan matematika kelas XI IPA 3 adalah 77
