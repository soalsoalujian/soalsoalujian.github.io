34. soal dari buku detik-detik UN matematika SMA IPA kategori : trigonometri soal nomor 28


Kelas            : 11
Mapel           : Matematika
Kategori      : Trigonometri lanjut
Kata kunci  : trigonometri, aturan cosinus
Kode            : 11.2.3 (Kelas 11 Matematika Bab 3-Trigonometri lanjut)

Gunakan aturan cos untuk segitiga ABD:
BD²=AB²+AD²-2.AB.AD. cos A

Gunakan aturan cos untuk segitiga BCD:
BD²=BC²+CD²-2.BC.CD. cos C

Berdasarkan sifat segiempat tali busur, sudut A dan C berjumlah 180° (karena berhadapan)

BD²=BD²
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²-2.BC.CD. cos C
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²-2.BC.CD. cos (180°-A)
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²+2.BC.CD. cos A
AB²+AD²-BC²-CD² = 2.BC.CD. cos A+2.AB.AD. cos A
AB²+AD²-BC²-CD² = cos A (2.BC.CD+2.AB.AD)
cos A = (AB²+AD²-BC²-CD²) / (2.BC.CD+2.AB.AD)
cos A = (4²+6²-3²-3²) / (2.3.3+2.4.6)
cos A = (16+36-9-9) / (18+48)
cos A = 34/66
cos A= 17/33

Jadi, cos ∠BAD adalah 17/33

[tex]\:[/tex]

Pendahuluan

Integral adalah invers atau kebalikan dari operasi diferensial/turunan, Oleh karena itu integral juga dapat disebut sebagai anti turunan, Berdasarkan bentuknya integral dibagi menjadi 2 jenis yaitu; Integral tentu dan Integral tak tentu, Ciri khas dari Integral tentu adalah memiliki batas atas dan bawah, Sedangkan Integral tak tentu sama sekali tidak memiliki batas atas maupun bawah.

[tex]\:[/tex]

Berikut beberapa sifat integral tak tentu

[tex]\displaystyle{\sf \purple{ \int dx } ~\rightarrow~ \blue{x + C}}[/tex]

[tex]\displaystyle{\sf \purple{ \int mdx } ~\rightarrow~\blue{mx + C}}[/tex]

[tex]\displaystyle{\sf \purple{ \int x^{-1}dx } ~\rightarrow~\blue{In|x| + C}}[/tex]

[tex]\displaystyle{\sf \purple{ \int o(f(x))dx} ~\rightarrow~ \blue{o \int f(x)dx}}[/tex]

[tex]\displaystyle{\sf \purple{ \int [f(x) \pm g(x)] dx} ~\rightarrow~ \blue{\int f(x)dx \pm \int g(x)dx }}[/tex]

[tex]\displaystyle{\sf \purple{\int ax^n dx} ~\rightarrow~ \blue{ \frac{a}{n + 1}x^{n + 1} + C, Untuk ~n \not = 1}}[/tex]

[tex]\:[/tex]

Berikut beberapa sifat integral tentu

[tex]\displaystyle{\sf \purple{\int\limits_{r}^r f(x)dx} ~\rightarrow~ \blue{0}}[/tex]

[tex]\displaystyle{\sf \purple{\int\limits_{o}^u f(x)dx} ~\rightarrow~ \blue{- \int\limits_{u}^o f(x)dx}}[/tex]

[tex]\displaystyle{\sf \purple{ \int\limits_{r}^i [f(x) \pm g(x)] dx} ~\rightarrow~ \blue{\int\limits_{r}^i f(x)dx \pm \int\limits_{r}^i g(x)dx }}[/tex]

[tex]\displaystyle{\sf \purple{\int\limits_{r}^a nf(x)dx} ~\rightarrow~ \blue{n \int\limits_{r}^a f(x)dx}}[/tex]

[tex]\:[/tex]

[tex]\:[/tex]

Soal

Penyelesaian dari [tex] \displaystyle{\sf \int {3}^{2} + 2x - {4x}^{3} }[/tex] adalah ...

[tex]\:[/tex]

DiketahuiBentuk integral tak tentu → [tex] \displaystyle{\sf \int {3}^{2} + 2x - {4x}^{3} }[/tex]

[tex]\:[/tex]

Ditanya

Penyelesaian

[tex]\:[/tex]

☆ Solusi

Gunakan sifat integral bahwa:

[tex]\displaystyle{\sf \purple{\int ax^n dx} ~\rightarrow~ \blue{ \frac{a}{n + 1}x^{n + 1} + C}}[/tex]

[tex]\:[/tex]

Maka:

[tex]~\rightarrow~ \displaystyle{\sf \int {3}^{2} + 2x - {4x}^{3} }[/tex]

[tex]~\rightarrow~\displaystyle{\sf \int 3^2 dx + \int 2xdx + \int - 4x^3dx}[/tex]

[tex]~\rightarrow~\displaystyle{\sf 3^2 \int 1dx + 2 \int xdx - \int 4x^3dx}[/tex]

[tex]~\rightarrow~\displaystyle{\sf 3^2x + \left( 2 \times \frac{1}{1 + 1}x^{1 + 1} \right) - \left( 4 \int x^2dx \right)}[/tex]

[tex]~\rightarrow~\displaystyle{\sf 9x + \left( 2 \times \frac{1}{2}x^{2} \right) - \left( 4 \times \frac{1}{3 + 1}x^{3 + 1} \right)}[/tex]

[tex]~\rightarrow~\displaystyle{\sf 9x + \left( \frac{2 \times 1}{2}x^{2} \right) - \left( \frac{4 \times 1}{4}x^{4} \right)}[/tex]

[tex]~\rightarrow~\displaystyle{\sf 9x + \left( \frac{2}{2}x^{2} \right) - \left( \frac{4}{4}x^{4} \right)}[/tex]

[tex]~\rightarrow~\displaystyle{\sf 9x + 1x^{2} - 1x^{4} }[/tex]

[tex]~\rightarrow~\displaystyle{\sf 9x + x^{2} - x^{4} }[/tex]

Karena merupakan bentuk integral tak tentu, Tambahlan konstanta ( + C ), Sehingga diperoleh:

[tex]\boxed{\blue{~\rightarrow~\displaystyle{\sf 9x + x^{2} - x^{4} + C}}}[/tex]

[tex]\:[/tex]

• Kesimpulan

Jadi, Penyelesaian dari [tex] \displaystyle{\sf \int {3}^{2} + 2x - {4x}^{3} }[/tex] adalah 9x + x² - x⁴ + C
