1. Nomor 1.Matematika statistika kelas 11


Diketahui data sebagai berikut:
51,5  45,5  52,5  63,5  50,5  58,5  61,5  88,5  70,5  77,5  81,5  63,5
n = 12
Data terurutnya:
45,5  50,5  51,5  52,5  58,5  61,5  63,5  63,5  70,5  77,5  81,5  88,5
a. Statistik lima serangkai
1) Data terkecil = 45,5
2) Data terbesar = 88,5
3) Median = Q2 = (61,5 + 63,5)/2 = 125/2 = 62,5
4) Kuartil pertama = Q1 = (51,5 + 52,5)/2 = 104/2 = 52
5) Kuartil ketiga = Q3 = (70,5 + 77,5)/2 = 148/2 = 74
b. Simpangan kuartil = 1/2 (Q3 – Q1) = ½ (74 – 52) = ½ (22) = 11
c. Letak desil ke-i = i(n + 1)/10
Letak desil ke-3 = 3(12 + 1)/10 = 3(13)/10 = 39/10 = 3,9
D3 = x3 + 0,9(x4 – x3)
      = 51,5 + 0,9(52,5 – 51,5)
      = 51,5 + 0,9(1)
      = 51,5 + 0,9 = 52,4
Letak desil ke-6 = 6(12 + 1)/10 = 6(13)/10 = 78/10 = 7,8
D6 = x7 + 0,8(x8 – x7)
      = 63,5 + 0,8(63,5 – 63,5)
      = 63,5 + 0,8(0) = 63,5

Letak desil ke-8 = 8(12 + 1)/10 = 8(13)/10 = 104/10 = 10,4
D8 = x10 + 0,4(x11 – x10)
      = 77,5 + 0,4(81,5 – 77,5)
      = 77,5 + 0,4(4)
      = 77,5 + 1,6 = 79,1

36. Soal statistika , ada yang bisa bantu ?


kedua kelompok masing2 beranggotakan 5 orang.
[tex]rata - rata = \frac{jumlah \: data}{banyak \: data} [/tex]
[tex]jumlah \: data = banyak \: data \times rata2[/tex]
jumlah nilai kelompok 1 = 5 x 5,00 = 25
jumlah nilai kelompok 2 = 5 x 5,8 = 29
misal nilai yg ditukar dari klomok 1 adalah a dan dari klompok 2 adalah b.
ketika ditukar, jumlah nilai klompok 1 berkurang a dan bertambah b, sedangkan jumlah nilai klompok 2 akan berkurang b dan bertambah a, kemudian rata2 jadi sama, maka:
[tex] \frac{25 - a + b}{5} = \frac{29 - b + a}{5} [/tex]
[tex]25 - a + b = 29 - b + a[/tex]
2b - 2a = 29 - 25
2(b - a) = 4
b - a = 2
jadi, selisih nilai kedua anak yg ditukar adalah 2
