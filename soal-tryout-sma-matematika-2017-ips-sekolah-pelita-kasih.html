1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a

=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}
