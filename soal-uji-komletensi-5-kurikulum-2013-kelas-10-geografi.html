7. Ada yang bisa Matematika kurikulum 2013 uji kopetensi 2.2 bab 2 kelas 10 /X


Tentukan himpunan penyelesaian persamaan berikut !

Pembahasan :

1) |x - 2| = 6

• x - 2 = 6
=> x = 8

• -(x - 2) = 6
=> -x + 2 = 6
=> -x = 4
=> x = -4

HP = {-4, 8}



2) |3x - 5| = 7

• 3x - 5 = 7
=> 3x = 12
=> x = 4

• -(3x - 5) = 7
=> -3x + 5 = 7
=> -3x = 2
=> x = -2/3

HP = {-2/3, 4}



3) |x| + |x - 5| = 7
|x| = x jika x ≥ 0
|x| = -x jika x < 0
|x - 5| = (x - 5) jika x ≥ 5
|x - 5| = (-x + 5) jika x < 5

• jika x < 0
|x| + |x - 5| = 7
-x + (-x + 5) = 7
-2x = 2
x = -1 ..... (memenuhi syarat x < 0)

• jika 0 ≤ x < 5
|x| + |x - 5| = 7
x + (-x + 5) = 7
5 = 7
(Salah)

• jika x ≥ 5
|x| + |x - 5| = 7
x + (x - 5) = 7
2x = 12
x = 6 ...... (memenuhi syarat x ≥ 5)

Jadi HP = {-1, 6}


4) |2x - 2| + |3x - 8| = 5
|2x - 2| = (2x - 2) jika x ≥ 1
|2x - 2| = (-2x + 2) jika x < 1
|3x - 8| = (3x - 8) jika x ≥ 8/3
|3x - 8| = (-3x + 8) jika x < 8/3

• jika x < 1
|2x - 2| + |3x - 8| = 5
(-2x + 2) + (-3x + 8) = 5
-5x + 10 = 5
-5x = -5
x = 1 ...... (tidak memenuhi syarat x < 1)

• jika 1 ≤ x < 8/3
|2x - 2| + |3x - 8| = 5
(2x - 2) + (-3x + 8) = 5
-x + 6 = 5
-x = -1
x = 1 ..... (memenuhi syarat 1 ≤ x < 8/3)

• jika x ≥ 8/3
|2x - 2| + |3x - 8| = 5
(2x - 2) + (3x - 8) = 5
5x - 10 = 5
5x = 15
x = 3 ...... (memenuhi syarat x ≥ 8/3)

Jadi HP = {1, 3}



5) |x - 1| + |2x| + |3x + 1| = 6
|x - 1| = (x - 1) jika x ≥ 1
|x - 1| = (-x + 1) jika x < 1
|2x| = (2x) jika x ≥ 0
|2x| = (-2x) jika x < 0
|3x + 1| = (3x + 1) jika x ≥ -1/3
|3x + 1| = (-3x - 1) jika x < -1/3

• jika x < -1/3
|x - 1| + |2x| + |3x + 1| = 6
(-x + 1) + (-2x) + (-3x - 1) = 6
-6x = 6
x = -1 ...... (memenuhi syarat x < -1/3)

• jika -1/3 ≤ x < 0
|x - 1| + |2x| + |3x + 1| = 6
(-x + 1) + (-2x) + (3x + 1) = 6
2 = 6
(salah)

• jika 0 ≤ x < 1
|x - 1| + |2x| + |3x + 1| = 6
(-x + 1) + (2x) + (3x + 1) = 6
4x + 2 = 6
4x = 4
x = 1 ...... (Tidak memenuhi syarat 0 ≤ x < 1)

• jika x ≥ 1
|x - 1| + |2x| + |3x + 1| = 6
(x - 1) + 2x + (3x + 1) = 6
6x = 6
x = 1 ...... (memenuhi syarat x ≥ 1)

Jadi HP = {1, -1}

10. uji kompetensi hal 173s.d.176.kelas 9 kurikulum 2013​


Jawaban:

1. Benda yang memiliki beda potensial yang lebih besar adalah benda pada gambar C.

2. Arah loncatan elektron adalah dari benda yang titik potensialnya rendah ke benda yang titik potensialnya lebih tinggi.

3. Apabila benda A dan benda B memiliki elektron yang sama, yang terjadi adalah tidak ada perpindahan elektron, karena tidak ada perbedaan potensial antara kedua benda.

Pembahasan

ARUS LISTRIK KONVENSIONAL

Arus listrik mengalir pada sebuah rangkaian listrik yang tertutup. Aliran ini diakibatkan adanya perbedaan potensial antara kedua kutubnya. Aliran muatan listrik positif mengalir dari tempat yang memiliki muatan listrik positif lebih besar (atau potensial tinggi) ke tempat yang bermuatan listrik positif lebih kecil (atau potensial rendah). Arus listrik inilah yang disebut sebagai arus listrik konvensional.

Setelah penemuan elektron oleh J.J. Thomson, diketahui, sebenarnya yang mengalir adalah elektron. Arah aliran elektron, kebalikan dari arah muatan listrik positif. Atah aliran elektron dari potensial rendah ke potensial tinggi.

Apabila tidak ada perbedaan potensial, arus listrik, baik muatan positif dan elektron, tidak akan mengalir.

Diketahui:

1. A. A = 2+ B = 2-

B. A = 2+ B = 4+

C. A = 1- B = 8-

D. A = 2- B = 2-

Ditanyakan:

1. Beda potensial terbesar ?

Penjelasan:

1. Beda potensial terbesar apa bila selisih antara muatan positif di kedua benda semakin besar.

A. Selisih = 2 - (-2) = 2 + 2 = 4

Arah arus dari A ke B.

B. Selisih = 4 - 2 = 2

Arah arus dari B ke A.

C. Selisih = -1 - (-8) = -1 + 8 = 7

Arah arus dari A ke B.

D. Selisih = -2 - (-2) = -2 + 2 = 0

Arus tidak mengalir.

Benda yang memiliki beda potensial yang lebih besar adalah benda pada gambar C.

2. Arah loncatan elektron adalah dari benda yang titik potensialnya rendah ke benda yang titik potensialnya lebih tinggi.

3. Apabila benda A dan benda B memiliki elektron yang sama, yang terjadi adalah tidak ada perpindahan elektron, karena tidak ada perbedaan potensial antara kedua benda.

1.    Berikan contoh tindakan ekonomi yang didasarkan atas kejujuran, tanggung jawab dan santun!
Jawab : 
            a. Kejujuran : Seorang pedagang yang menjual dagangannya dengan takaran yang tepat.
            b. Tanggung Jawab : Seorang petani yang mengolah, menanam, dan memanen hasil pertaniannya.
            c. Santun : Seorang polisi yang mengayomi dan memberikan tauladan yang baik bagi masyarakat.

2.    Uraikan tiga pokok kegiatan ekonomi! 
Jawab :
            a. Kegiatan Produksi, merupakan kegiatan untuk menghasilkan barang/jasa, untuk memperoleh keuntungan, dan memenuhi kebutuhan konsumen.
            b. Kegiatan Distribusi, merupakan kegiatan menyalurkan barang dari produsen kepada konsumen. Sistem distribusi: distribusi langsung, distribusi semi langsung, dan distribusi tidak langsung.
            c. Kegiatan Konsumsi, merupakan kegiatan memakai, menggunakan atau menghabiskan kegunaan barang/jasa baik secara berangsur-angsur atau sekaligus habis.

3.    Tuliskan lima macam kegiatan yang kamu lakukan yang berdasar pada prinsip ekonomi! 
Jawab :
            a. Membeli barang dengan memilih terlebih dahulu.
            b. Membeli barang dengan kualitas bagus.
            c. Membeli barang sesuai dengan yang direncanakan.
            d. Membeli barang bukan berdasarkan keinginan melainkan kebutuhan.
            e. Setiap awal bulan membuat daftar kebutuhan berdasar skala prioritas.

4.    Sebutkan  4 macam motif ekonomi! 
Jawab :
            a. Motif untuk memenuhi kebutuhan.
            b. Motif untuk berbuat sosial.
            c. Motif untuk mendapatkan penghargaan.
            d. Motif untuk memperoleh kekuasaan.

5.    Mengapa dalam melakukan tindakan ekonomi harus berpedoman pada prinsip ekonomi! 
Jawab : 
            Ada beberapa alasan mengapa setiap tindakan ekonomi harus berpedoman pada prinsip ekonomi, yaitu guna mendapatkan hasil tertentu dengan pengorbanan yang sekecil mungkin, memastikan pengorbanan yang dilakukan sesuai dengan hasil yang diharapkan, serta mendapatkan hasil yang maksimal dengan kemampuan dan alat yang tersedia.

6.    Apa yang dimaksud dengan manusia sebagai makhluk ekonomi! 
Jawab : 
            Maksudnya yaitu bahwa manusia selalu ingin memenuhi kebutuhan yang berdasarkan cara-cara rasional sehingga diperoleh kesejahteraanya.

7.    Berikan contoh tindakan yang kamu lakukan sebagai makhluk ekonomi dan sebutkan pula tujuan kamu melakukan tindakan itu!

8.    Apa yang dimaksud dengan manusia sebagai makhluk sosial! 
Jawab : 
            Makhluk yang selalu hidup bersama dan tidak dapat hidup sendiri dalam memenuhi kebutuhannya.

9.    Berikan contoh minimal 2 tindakan kamu sebagai makhluk sosial! 
Jawab :

            b. Berperilaku sopan dan santun terhadap siapapun.

32. tolong jawab soal kelas 10 kurikulum 2013 halaman 73 sampai 75.


Pembahasan

Diminta untuk menjawab soal buku cetak kelas 10 kurikulum 2013 halaman 73 sampai 75.

Ini merupakan persoalan struktur atom terkait perkembangan model atom. Sejak dicetuskannya konsep atom, beberapa percobaan telah dilakukan untuk meneliti partikel penyusun atom. Berikut beberapa percobaan tersebut:

[1]. Model atom Thomson

Melalui percobaan sinar katoda, Thomson mengemukakan bahwa atom adalah bola padat bermuatan positif dan di dalamnya tersebar elektron yang bermuatan negatif. Model ini dikenal sebagai model roti kismis. Kelemahan model ini adalah belum ditemukannya inti atom.

[2]. Model atom Rutherford

Dari percobaan hamburan sinar alfa terhadap lempeng emas tipis, Rutherford menyimpulkan:

atom terdiri atas inti atom yang bermuatan positif dan elektron-elektron bermuatan negatif yang beredar mengeliling inti;massa atom terpusat pada inti atom;sebagian besar bagian atom merupakan ruang hampa;karena atom bersifat netral, jumlah elektron di dalam atom sama dengan jumlah muatan inti atom.

Kelemahan model ini adalah Rutherford tidak dapat menjelaskan mengapa elektron tidak jatuh ke dalam inti atom. Sebab gerakan elektron mengelilingi inti disertai pemancaran energi, sehingga energi elektron semakin berkurang dan lintasannya makin mendekati inti dan kemudian jatuh ke dalam inti.

[3]. Model atom Bohr

Setelah melakukan percobaan dan mengamati spektrum hidrogen, berikut kesimpulan Bohr:

atom terdiri dari inti yang bermuatan positif dan dikelilingi elektron bermuatan negatif di dalam suatu lintasan tertentu atau tingkat energi yang disebut kulit atom;elektron dapat berpindah dari satu lintasan ke lintasan lain dengan menyerap atau memancarkan energi. Jika elektron berpindah ke lintasan yang lebih tinggi maka elektron akan menyerap energi, dan jika elektron berpindah ke lintasan yang lebih rendah maka elektron akan memancarkan energi.

Kelemahan model ini adalah Bohr tidak dapat menerangkan spektrum garis elektron banyak, serta tak adapat menjelaskan adanya modifikasi pengaruh medan magnet dalam atom hidrogen.
