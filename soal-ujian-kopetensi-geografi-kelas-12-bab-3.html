3. Ada yang bisa Matematika kurikulum 2013 uji kopetensi 2.2 bab 2 kelas 10 /X


Tentukan himpunan penyelesaian persamaan berikut !

Pembahasan :

1) |x - 2| = 6

• x - 2 = 6
=> x = 8

• -(x - 2) = 6
=> -x + 2 = 6
=> -x = 4
=> x = -4

HP = {-4, 8}



2) |3x - 5| = 7

• 3x - 5 = 7
=> 3x = 12
=> x = 4

• -(3x - 5) = 7
=> -3x + 5 = 7
=> -3x = 2
=> x = -2/3

HP = {-2/3, 4}



3) |x| + |x - 5| = 7
|x| = x jika x ≥ 0
|x| = -x jika x < 0
|x - 5| = (x - 5) jika x ≥ 5
|x - 5| = (-x + 5) jika x < 5

• jika x < 0
|x| + |x - 5| = 7
-x + (-x + 5) = 7
-2x = 2
x = -1 ..... (memenuhi syarat x < 0)

• jika 0 ≤ x < 5
|x| + |x - 5| = 7
x + (-x + 5) = 7
5 = 7
(Salah)

• jika x ≥ 5
|x| + |x - 5| = 7
x + (x - 5) = 7
2x = 12
x = 6 ...... (memenuhi syarat x ≥ 5)

Jadi HP = {-1, 6}


4) |2x - 2| + |3x - 8| = 5
|2x - 2| = (2x - 2) jika x ≥ 1
|2x - 2| = (-2x + 2) jika x < 1
|3x - 8| = (3x - 8) jika x ≥ 8/3
|3x - 8| = (-3x + 8) jika x < 8/3

• jika x < 1
|2x - 2| + |3x - 8| = 5
(-2x + 2) + (-3x + 8) = 5
-5x + 10 = 5
-5x = -5
x = 1 ...... (tidak memenuhi syarat x < 1)

• jika 1 ≤ x < 8/3
|2x - 2| + |3x - 8| = 5
(2x - 2) + (-3x + 8) = 5
-x + 6 = 5
-x = -1
x = 1 ..... (memenuhi syarat 1 ≤ x < 8/3)

• jika x ≥ 8/3
|2x - 2| + |3x - 8| = 5
(2x - 2) + (3x - 8) = 5
5x - 10 = 5
5x = 15
x = 3 ...... (memenuhi syarat x ≥ 8/3)

Jadi HP = {1, 3}



5) |x - 1| + |2x| + |3x + 1| = 6
|x - 1| = (x - 1) jika x ≥ 1
|x - 1| = (-x + 1) jika x < 1
|2x| = (2x) jika x ≥ 0
|2x| = (-2x) jika x < 0
|3x + 1| = (3x + 1) jika x ≥ -1/3
|3x + 1| = (-3x - 1) jika x < -1/3

• jika x < -1/3
|x - 1| + |2x| + |3x + 1| = 6
(-x + 1) + (-2x) + (-3x - 1) = 6
-6x = 6
x = -1 ...... (memenuhi syarat x < -1/3)

• jika -1/3 ≤ x < 0
|x - 1| + |2x| + |3x + 1| = 6
(-x + 1) + (-2x) + (3x + 1) = 6
2 = 6
(salah)

• jika 0 ≤ x < 1
|x - 1| + |2x| + |3x + 1| = 6
(-x + 1) + (2x) + (3x + 1) = 6
4x + 2 = 6
4x = 4
x = 1 ...... (Tidak memenuhi syarat 0 ≤ x < 1)

• jika x ≥ 1
|x - 1| + |2x| + |3x + 1| = 6
(x - 1) + 2x + (3x + 1) = 6
6x = 6
x = 1 ...... (memenuhi syarat x ≥ 1)

Jadi HP = {1, -1}

Jawaban uji kopetensi 6 matematika kelas 8 semester 2 hal 45

Teorama Pythagoras adalah rumus untuk mencari sisi-sisi pada segitiga siku-siku

Bunyi Teorema Pythagoras adalah Kuadrat sisi miring sama dengan jumlah kuadrat kedua sisi penyikunya

Sisi miring / Hipotenusa biasanya sisi yang terpanjang diantara sisi-sisi lainnya

Pembahasan :

1. Diketahui segitiga KLM dengan panjang sisi-sisinya k, l, dan m.

Pernyataan berikut yang benar dari segitiga KLM adalah ....

a. Jika m² = l² + k², besar ∠K = 90°

b. Jika m² = l² − k², besar ∠M = 90°

c. Jika m² = k² − l², besar ∠L = 90°

d. Jika k² = l² + m², besar ∠K = 90° (Benar)

Diketahui :

Segitiga KLM dengan panjang sisi k, l dan m

Ditanya :

Pernyataan yang benar ?

Dijawab :

Lihat gambar ilustrasi

a. Jika m² = l² + k², besar ∠K = 90°

Apabila ∠K = 90° maka sisi miring adalah sisi k

maka menurut Rumus Pythagoras :

k² = l² + m² (Pernyataan salah)

b. Jika m² = l² − k², besar ∠M = 90°

Apabila ∠M = 90° maka sisi miring adalah sisi m

maka menurut Rumus Pythagoras :

m² = k² + l² (Pernyataan salah)

c. Jika m² = k² − l², besar ∠L = 90°

Apabila ∠L = 90° maka sisi miring adalah sisi l

maka menurut Rumus Pythagoras :

l² = k² + m² (Pernyataan salah)

D. Jika k² = l² + m², besar ∠K = 90°

Apabila ∠K = 90° maka sisi miring adalah sisi k

maka menurut Rumus Pythagoras :

k² = l² + m² (Pernyataan benar)


2. Perhatikan gambar berikut. Panjang sisi PQ = ... cm.

a. 10      c. 13

b. 12      d. 14

Diketahui :

PR = 26cm

QR = 24cm

Ditanya :

PQ ?

Dijawab :

PQ² + QR² = PR²

PQ² + 24² = 26²

PQ² + 576 = 676

PQ² = 676 - 576

PQ = √100 = 10 cm (A)


3. Diketahui kelompok tiga bilangan berikut.

(i) 3, 4, 5          (iii) 7, 24, 25

(ii) 5, 13, 14      (iv) 20, 21, 29

Kelompok bilangan di atas yang merupakan tripel Pythagoras adalah ....

a. (i), (ii), dan (iii)          c. (ii) dan (iv)

b. (i) dan (iii)                  d. (i), (ii), (iii), dan (iv)

Diketahui :

kelompok tiga bilangan berikut.

(i) 3, 4, 5         (iii) 7, 24, 25

(ii) 5, 13, 14     (iv) 20, 21, 29

Ditanya :

Kelompok bilangan diatas yang merupakan Triple Pythagoras ?

Dijawab :

(i) 3, 4, 5    

sisi miring = 5

5² = 3² + 4²

25 = 9 + 16

25 = 25 (Terbukti)    

(ii) 5, 13, 14  

Sisi miring = 14

14² = 5² + 13²

196 = 25 + 169

196 ≠ 194 (Tidak terbukti)

(iii) 7, 24, 25

Sisi miring = 25

25² = 7² + 24²

625 = 49 + 576

625 = 625 (Terbukti)

(iv) 20, 21, 29

Sisi miring = 29

29² = 20² + 21²

841 = 400 + 441

841 = 841 (Terbukti)

Jadi yang merupakan triple pythagoras adalah (i), (III) dan (iv) (B)


4. (i) 3 cm, 5 cm, 6 cm       (iii) 16 cm, 24 cm, 32 cm  

(ii) 5 cm, 12 cm, 13 cm       (iv) 20 cm, 30 cm, 34 cm

Ukuran sisi yang membentuk segitiga lancip ditunjukkan oleh ....

a. (i) dan (ii)         c. (ii) dan (iii)

b. (i) dan (iii)        d. (iii) dan (iv)

Diketahui :

(i) 3 cm, 5 cm, 6 cm          (iii) 16 cm, 24 cm, 32 cm  

(ii) 5 cm, 12 cm, 13 cm       (iv) 20 cm, 30 cm, 34 cm

Ditanya :

Ukuran sisi yang merupakan segitiga lancip adalah ?

Dijawab :

Persamaan sisi segitiga :

c = sisi miring

c² > a² + b² (Segitiga tumpul)

c² = a² + b² (Segitiga siku-siku)

c² < a² + b² (Segitiga lancip)

(i).   3 cm , 5 cm, 6 cm

c = 6cm

6² > 3² + 5²

36 > 9 + 25

36 > 34  

segitiga tumpul, karena c² > a² + b²

(ii).  5 cm , 12 cm, 13 cm

c = 13cm

13²  = 5² + 12²

169 = 25 + 144

169 = 169

Segitiga siku-siku, karena c² = a² + b²

(iii).  16 cm , 24 cm, 32 cm

c = 32cm

32² > 16² + 24²

1024 > 256 + 576

1024 > 832

Segitiga tumpul, karena c² > a² + b²

(iv).  20 cm , 30 cm, 34 cm

c = 34cm

34² < 20² + 30²

1156 < 400 + 900

1156 < 1300

Segitiga lancip, karena c² < a² + b²

Yang merupakan segitiga lancip adalah (iv) (Tidak ada jawaban)

10. Jawaban uji kopetensi bab 2 kls 7 ipa hal 44


Jika para petani melakukan pemberantasan serangga, maka populasi padi akan meningkat dan populasi katak akan menurun. Factor yang mungkin mempengaruhi ekosistem sawah antara lain bencana alam seperti misalnya banjir atau kekeringan. Komponen abiotik pada ekosistem sawah antara lain tanah, air, udara, batu, cahaya matahari.


Pembahasan

Berikut merupakan jawaban uji kompetensi bab 2 dari buku IPA kelas VII semester 2 mulai halaman 44.

Poin nomor 1

Jika para petani melakukan pemberantasan serangga, maka populasi padi akan meningkat dan populasi katak akan menurun.

Poin nomor 2

Factor yang mungkin mempengaruhi ekosistem sawah antara lain bencana alam seperti misalnya banjir atau kekeringan yang akan mematikan padi sebagai produsen. Penggunaan pestisida oles para petani juga dapat mempengaruhi keseimbangan ekosistem karena matinya serangga.

Komponen abiotik pada ekosistem sawah antara lain tanah, air, udara, batu, cahaya matahari.

Poin nomor 3

Jika produsen dalam ekosistem sawah rusak karena bencana alam misalnya banjir, maka serangga sebagai konsumen primer juga akan mengalami kekurangan makan sehingga akan banyak yang mati, demikian juga katak sebagai konsumen sekunder akan kekurangan makan karena serangga banyak yang mati. Ular sebagai konsumen ketiga juga akan banyak yang mati karena katak yang merupakan makanan ular banyak yang mati. Elang sebagai konsumen terakhir juga akan banyak yang mati karena ular sebagai makanan elang banyak yang mati. Kesimpulannya, jika produsen punah. Maka akan diikuti oleh kepunahan semua makhuk hidup pada rantai makanan tersebut.

Poin nomor 4

Padi, serangga, katak, ular dan elang dapat hidup bersama di ekosistem sawah karena terjadi suatu perpindahan energy dalam bentuk rantai makanan yang menyediakan makanan untuk bertahan hidup.

Poin nomor 5

Jika pada ekosistem sawah tidak ada ular, maka populasi katak akan sangat banyak dan jumlahnya menjadi tidak dapat dikendalikan.

Jika pada ekosistem sawah tidak ada ular, maka populasi elang akan punah karena elang mendapatkan energy untuk hidup dengan memakan ular.

Dampak yang ditimbulkan akibat aktivitas pertanian pada ekosistem sawah misalnya penyemprotan serangga adalah rusaknya ekosistem sawah karena salah satu anggota rantai makanan jumlahnya menurun atau jumlahnya menjadi tidak seimbang pada rantai makanan di ekosistem tersebut.

Poin nomor 6

Pabrik tersebut mengeluarkan gas buang dari cerobong asap berupa asap yang tebal, pekat dan berwarna hitam. Gas buang tersebut akan menyebabkan terjadi nya polusi udara di sekitar pabrik. Jika asap pabrik tersebut mengandung zat kimia berbahaya maka akan menimbulkan dampak buruk bagi kesehatan masyarakat di sekitarnya, misalny bila kena mata, maka akan menimbulkan iritasi mata.

Jika diperhatikan pada gambar denah yang tertera pada soal, dapat kita lihat bahwa arah angin bertiup dari barat ke timur sehingga gas buang dari pabrik otomatis akan mengarah ke timur sehingga masyarakat di sebelah timur pabrik akan mengalami iritasi mata sebagai dampak gas buang dari cerobong asap pabrik tersebut.

Poin nomor 7

A merupakan oksigen yang dihasilkan oleh tumbuhan hijau atau alga sebagai produsen pada proses fotosintesis.

B merupakan karbon dioksida yang merupakan hasil respirasi hewan sebagai konsumen.

Gambar tersebut dapat mewakili hubungan saling ketergantungan karena tumbuhan hijau atau alga memerlukan karbon dioksida sebagai bahan dasar fotosintesis dan akhirnya dapat menghasilkan oksigen, sedangkan hewan-hewan memerlukan oksigen untuk bernapas atau respirasi dan menghasilkan zat sisa berupa karbon dioksida yang dapat dipakai untuk fotosintesis oleh tumbuhan.

16. Bantu dlu broo soal. Geografi uji komptensi bab 5


Hai, sahabat Brainly!

Yuk, kita coba selesaikan satu per satu ya.

Pembahasan

[No.1]: Cabang ilmu Geografi yang mempelajari seluk-beluk kependudukan disebut E. Geografi penduduk

Alasan: Geografi penduduk adalah ilmu yang mempelajari aspek kependudukan berdasarkan pendekatan keruangan, kelingkungan dan kewilayahan.

***

[No.2]: Peristiwa terlepasnya bayi dari rahim yang ditandai dengan adanya tanda-tanda kehidupan disebut B. Fertilitas.

Alasan: Fertilitas sering disebut juga natalitas atau lahir hidup  (live birth), yaitu kelahiran bayi dengan tanda hidup seperti bernapas, berteriak, jantung berdenyut dan lain-lain.

***

[No.3]: Angka kelahiran umum wilayah tersebut adalah C. 40 jiwa

Alasan:

Diketahui: Kelahiran (B) = 400 bayi

Jumlah wanita usia 15-49 tahun = 10.000 jiwa

→ Untuk menghitung angka kelahiran pada usia tersebut digunakan rumus:

[tex]ASBR = \frac{B_{x} }{P_{x}}\ x\ 1.000\\ASBR = \frac{B_{15\ -\ 49} }{P_{15\ -\ 49}}\ x\ 1.000\\ASBR = \frac{400}{10.000}\ x\ 1.000 \\ASBR = 0,04\ x\ 1.000\\ASBR = 40\ bayi\\[/tex]

Kesimpulannya diperoleh angka kelahiran menurut kelompok wanita umur 15-49 tahun adalah 40 bayi. Artinya, dari setiap 1.000 penduduk wanita umur 15-49 tahun di wilayah tersebut terdapat kelahiran 40 bayi.

***

[No.4]: Gerak penduduk ke suatu daerah tanpa ada tujuan untuk menginap disebut B. Sirkulasi

Alasan: Migrasi sirkulasi biasanya disebut juga commuter atau migrasi penduduk pulang-pergi yang tidak bertujuan untuk menetap. Contohnya, seseorang bekerja di Jakarta dan pulang ke Bekasi di hari yang sama.

***

[No.5]: Faktor penarik urbanisasi adalah B. Di kota banyak fasilitas kehidupan.

Alasan: Faktor penarik urbanisasi berarti faktor yang berasal dari kota. Sedangkan faktor pendorong berarti faktor yang berasal dari desa. Jangan lupa ya! :)

***

[No.6]: Transmigrasi yang biayanya ditanggung antara pemerintah dan transmigran yaitu transmigrasi C. Sektoral

Alasan: Transmigrasi sektoral adalah transmigrasi yang dibedakan berdasarkan pembiayaannya, yakni dibiayai bersama-sama oleh pemerintah dan transmigran.

***

[No.7]: Berikut ini merupakan karakteristik sensus penduduk, kecuali D. Bersifat pasif.

Alasan: Sensus penduduk memiliki 4 ciri khas yaitu bersifat individu, universal, serentak, dan periodik.

***

[No. 8]: Lembaga yang berwenang melaksanakan registrasi penduduk adalah C. Dinas kependudukan.

Alasan: Registrasi penduduk dilakukan pada setiap peristiwa kelahiran, kematian, pernikahan, perceraian, atau migrasi penduduk. Lembaga yang berwenang melakukan pencatatan berkala adalah dinas kependudukan di wilayah yang bersangkutan mulai dari tingkat desa hingga kabupaten dan provinsi. Selanjutnya data tersebut akan diteruskan ke Badan Pusat Statistik.

***

[No.9]: Angka rasio jenis kelamin di Kabupaten X adalah B. 88,75%

Alasan:

Diketahui: Jumlah penduduk laki-laki 35.500 jiwa

Jumlah penduduk perempuan 40.000 jiwa

→ Untuk menghitung rasio jenis kelamin digunakan rumus:

[tex]SR = \frac{Jumlah\ penduduk\ Laki-laki}{Jumlah\ penduduk\ perempuan}\ x\ 100\ persen\\SR = \frac{35.500}{40.000}\ x\ 100\ persen\\SR = 0,8875\ x\ 100\ persen\\SR = 88,75\ persen[/tex]

Jadi, angka rasio jenis kelamin di wilayah tersebut adalah 88,75%. Artinya, setiap 100 orang penduduk perempuan terdapat 88,75 atau 89 penduduk laki-laki.

***

[No.10]: Tujuan transmigrasi adalah C. Meratakan persebaran penduduk.

Alasan: Transmigrasi dilaksanakan karena adanya pemusatan penduduk di wilayah-wilayah tertentu, biasanya di kota-kota besar sehingga tidak adanya pemerataan penduduk.

1. Sebuah balok memiliki panjang 12 cm, lebar 8 cm dan tinggi 3 cm, serta berat sebesar 30 N. Jika balok tersebut diletakkan seperti gambar, maka :

Tekanan masing - masing sebesar :

2. Sebuah dongkrak hidrolik dapat mengankat benda dengan massa 1 ton serta luas penampang piston penghisap besar 0,2 m². Jika Luas penampang piston kecil 0,02 m², serta besar percepatan gravitasi 9,8 m/s², maka Gaya minimal yang harus diberikan agar dapat mengangkat benda adalah 980 N.


[tex]\boxed{\boxed{PEMBAHASAN}}[/tex]

[tex]\boxed{\boxed{BAB\:3.\:FLUIDA}}[/tex]


[tex]\boxed{Tekanan}[/tex]

Gaya merupakan sesuatu yang mengakibatkan benda bergerak.

Tekanan merupakan gaya per luas permukaan yang bersentuhan.

Persamaan tekanan adalah :

[tex]\boxed{p=F/A}[/tex]


Keterangan :

p : tekanan (N/m²)

F : gaya tekan (N)

A : Luas permukaan yang bersentuhan (m²)


[tex]\boxed{Hukum\:Pascal}[/tex]

Hukum Pascal Berbunyi :" Tekanan yang diberikan zat cair dalam ruang tertutup diteruskan ke segala arah dengan sama besar."

Persamaan hukum paskal adalah :

[tex]\boxed{\frac{F_{1} }{A_{1} }=\frac{F_{2} }{A_{2} } }[/tex]


====================

[tex]\boxed{\boxed{NOMOR\:1}}[/tex]

[tex]\boxed{Diketahui}[/tex]

Panjang (p) = 12 cm

lebar (l) = 8 cm

tinggi (t) = 3 cm

Berat balok (F) = 30 N


[tex]\boxed{Ditanya}[/tex]

Tekanan Balok masing - masing.


[tex]\boxed{Penyelesaian}[/tex]

Perhatikan Gambar !

   Tekanan Posisi sesuai gambar A

Pada gambar A, permukaan yang menyentuh lantai memiliki luas sebesar:

A = p x t

A = 12 cm x 3 cm

A = 36 cm²

atau

A = 36 x 10⁻⁴ m²

Jadi Luas permukaan yang bersentuhan adalah 36 x 10⁻⁴ m², sehingga tekanannya bisa dicari dengan menggunakan persamaan :

p = F / A

p = 30 N / 36 x 10⁻⁴ m²

p = 0,83 x 10⁴ N/m²

atau

p = 8,33 x 10³ N/m²

Jadi tekanan pada gambar A sebesar 8,33 x 10³ N/m²


   Tekanan posisi sesuai gambar B

Pada gambar B, permukaan yang menyentuh lantai memiliki luas sebesar:

A = t x l

A = 3 cm x 8 cm

A = 24 cm²

atau

A = 24 x 10⁻⁴ m²

Jadi Luas permukaan yang bersentuhan adalah 24 x 10⁻⁴ m², sehingga tekanannya bisa dicari dengan menggunakan persamaan :

p = F / A

p = 30 N / 24 x 10⁻⁴ m²

p = 1,25 x 10⁴ N/m²

atau

p = 12,5 x 10³ N/m²

Jadi tekanan pada gambar B sebesar 12,5 x 10³ N/m²


   Tekanan Posisi sesuai Gambar C

Pada gambar C, permukaan yang menyentuh lantai memiliki luas sebesar:

A = p x l

A = 12 cm x 8 cm

A = 96 cm²

atau

A = 96 x 10⁻⁴ m²

Jadi Luas permukaan yang bersentuhan adalah 96 x 10⁻⁴ m², sehingga tekanannya bisa dicari dengan menggunakan persamaan :

p = F / A

p = 30 N / 96 x 10⁻⁴ m²

p = 0,3125 x 10⁴ N/m²

atau

p = 3,1 x 10³ N/m²

Jadi tekanan pada gambar A sebesar 3,1 x 10³ N/m²


[tex]\boxed{Kesimpulan}[/tex]

Tekanan masing - masing sebesar :

   Gambar A : Tekanannya sebesar 8,33 x 10³ N/m²    Gambar B : Tekanannya sebesar 12,5 x 10³ N/m²    Gambar C : Tekanannya sebesar 3,1 x 10³ N/m²

======================

[tex]\boxed{\boxed{NOMOR\:2}}[/tex]

[tex]\boxed{Diketahui}[/tex]

Massa benda (m) = 1 ton = 1.000 kg

Luas penghisap besar (A₁) = 0,2 m²

Luas penghisap kecil (A₂) = 0,02 m²

Percepatan gravitasi (g) = 9,8 m/s²


[tex]\boxed{Ditanya}[/tex]

Gaya minimal yang diberikan.


[tex]\boxed{Penyelesaian}[/tex]

Besar gaya minimal yang diberikan bisa dicari dengan menggunakan persamaan :

[tex]\frac{F_{1} }{A_{1} }=\frac{F_{2} }{A_{2} }[/tex]

atau

[tex]F_{2} =F_{1} \frac{A_{2} }{A_{1} }[/tex]

dimana F₁ = m g, sehingga

[tex]F_{2} =mg \frac{A_{2} }{A_{1} }[/tex]

[tex]F_{2} =1.000kg\:9,8m/s^{2} \frac{0,02m^{2} }{0,2m^{2} }[/tex]

[tex]F_{2} =9.800 (0,1)\:N[/tex]

F₂ = 980 N

Jadi gaya minimum yang diberikan sebesar 980 N.


[tex]\boxed{Kesimpulan}[/tex]

Besar Gaya minimum yang diberikan sebesar 980 N.
