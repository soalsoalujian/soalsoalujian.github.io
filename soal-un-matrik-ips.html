15. bingung ngerjain soal matrik nya nih kak


Jawaban:

b. det(P) = -2 - x dan det(Q) = -6

Penjelasan dengan langkah-langkah:

a. Nilai x agar |P| = |Q|

penyelesaian:

(-2) - x = 0 - 6

-x = -6 + 2

-x = -4

x = 4

b. det (P) dan det (Q)

penyelesaian:

det (P) = ((-1) . 2) - (1 . x) = -2 - x

det (Q) = (0 .4) - (3 . 2) = 0 - 6 = -6

Menentukan determinan suatu matriks

[tex]\left[\begin{array}{ccc}a&b\\c&d\end{array}\right][/tex]

Ingat : det(P) = |P| = a · d - b · c

Diketahui matriks P = [tex]\left[\begin{array}{ccc}-1 & 1\\x & 2\end{array}\right][/tex] dan Q = [tex]\left[\begin{array}{ccc}0&3\\2&4\end{array}\right][/tex], Tentukanlah

a. Nilai agar |P| = |Q|

mencari nilai |Q| = a · d - b · c

                         = 0 · 4 - 3 · 2

                         = 0 - 6

                         = -6

nilai |P| = |Q|

-1 · 2 - 1 ·x = -6

-2 - x        = -6

-x             = -6 + 2

-x             = -4

x              = 4

Jawaban : x = 4
