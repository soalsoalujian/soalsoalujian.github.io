40. bantu lagi yha matematika tapi pake cara buat Unas ​


Jawab:

D. 3

Penjelasan dengan langkah-langkah:

langkah pertama : cari a

3a - 4b = -8

a = -8/3 + 4b/3

langkah kedua : masukkan a dan temukan b

2a + 3b = 23

2(-8/3 + 4b/3) + 3b = 23

-16/3 + 8b/3 + 3b = 23

semua dikali 3

= -16 + 8b + 9b = 69

17b = 69 + 16

17b = 85

b = 5

langkah ketiga : masukkan b dan temukan a

3a - 4b = -8

3a - 4(5) = -8

3a - 20 = -8

3a = -8 + 20

3a = 12

a = 4

langkah keempat : kerjakan yang ditanya

2a - b

= 2(4) - 5

= 8 - 5

= 3
