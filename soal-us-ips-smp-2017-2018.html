7. 3. Perhatikan Tabel data Produk Domestik Bruto Negara Alengka berikut :TAHUNPDB2016US$ 2.460 miliar2017US$ 2.495 miliar2018US$ 2.550 miliar2019US$ 2.665 miliarAnda Diminta menghitung laju pertumbuhan Ekonomi Tahun 2017, 2018 dan 2019.​


Besarnya laju pertumbuhan ekonomi tahun 2017 adalah 1,42%Besarnya laju pertumbuhan ekonomi tahun 2019 adalah 4,5%Besarnya laju pertumbuhan ekonomi tahun 2019 adalah 4,5%

.

PEMBAHASAN

Laju perrtumbuhan ekonomi ialah keadaan di suatu negara dimana mengalami perkembangan ekonomi dengan meningkatkan jumlah produksi barang/jasa dan juga kesejahteraan rakyatnya.

.

[tex]\large {\boxed{\boxed{\bold{Laju \:Pertumbuhan \:Ekonomi \:Tahun \:2017}}}}[/tex]

.

Diketahui :

[tex]\sf PDB_{t} = \$ 2.495 \:M \\ \\ \sf PDB_{t-1} = \$2.460 \:M[/tex]

Ditanyakan :

Laju pertumbuhan ekonomi (G) tahun 2017 ?

Penyelesaian :

[tex]{\boxed{\bold{G = \frac{PDB_{t} - PDB_{t-1} }{PDB_{t-1} } \times 100\%}}}[/tex]

[tex]\sf G = \frac{\$2.495 \:M - \$2.460 \:M}{\$2.460 \:M} \times 100\%[/tex]

[tex]\sf G = \frac{\$35 \:M}{\$2.460 \:M} \times 100\%[/tex]

[tex]\sf G = 1,42\%[/tex]

.

Jadi, besarnya laju pertumbuhan ekonomi tahun 2017 adalah 1,42%

.

[tex]\large {\boxed{\boxed{\bold{Laju \:Pertumbuhan \:Ekonomi \:Tahun \:2018}}}}[/tex]

.

Diketahui :

[tex]\sf PDB_{t} = \$ 2.550 \:M \\ \\ \sf PDB_{t-1} = \$2.495 \:M[/tex]

Ditanyakan :

Laju pertumbuhan ekonomi (G) tahun 2018 ?

Penyelesaian :

[tex]{\boxed{\bold{G = \frac{PDB_{t} - PDB_{t-1} }{PDB_{t-1} } \times 100\%}}}[/tex]

[tex]\sf G = \frac{\$2.550 \:M - \$2.495 \:M}{\$2.495 \:M} \times 100\%[/tex]

[tex]\sf G = \frac{\$ 55 \:M}{\$ 2.495 \:M} \times 100\% \\ \\ \sf G = 2,2\%[/tex]

.

Jadi, besarnya laju pertumbuhan ekonomi tahun 2018 adalah 2,2%

.

[tex]\large {\boxed{\boxed{\bold{Laju \:Pertumbuhan \:Ekonomi \:Tahun \:2019}}}}[/tex]

.

Diketahui :

[tex]\sf PDB_{t} = \$ 2.665 \:M \\ \\ \sf PDB_{t-1} = \$2.550 \:M[/tex]

Ditanyakan :

Laju pertumbuhan ekonomi (G) tahun 2019 ?

Penyelesaian :

[tex]{\boxed{\bold{G = \frac{PDB_{t} - PDB_{t-1} }{PDB_{t-1} } \times 100\%}}}[/tex]

[tex]\sf G = \frac{\$2.665 \:M - \$2.495 \:M}{\$2.550 \:M} \times 100\%[/tex]

[tex]\sf G = \frac{\$ 115 \:M}{\$ 2.550 \:M} \times 100\% \\ \\ \sf G = 4,5\%[/tex]

.

Jadi, besarnya laju pertumbuhan ekonomi tahun 2019 adalah 4,5%

.

40. Soal Latihan USBN 2017 / 2018 Mapel: Matematika (Peminatan) Satuan Pendidikan: SMA Tahun: 2017 / 2018 Kurikulum: Kurikulum 2013


Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5
